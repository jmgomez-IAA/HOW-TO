Problems

Pages are reformatted
Web pages designed for PC screens do not normally fit on a phone or Pocket PC. Opera therefore reformats pages when necessary to make them fit the screen. This function can be toggled on and off using Menu > Display > Fit to screen. Read more about display modes.
Out of memory
Make sure you have enough memory available for installing and browsing with Opera. Clear your device of all unnecessary images, applications, other large files and messages. Ensure that enough memory is allocated to running programs on your device, using Start > Settings > System > Memory. You may also wish to stop other applications that are running, using Start > Settings > System > Memory > Running Programs. For more information, please consult the help documentation for your device.
Out of memory while browsing
If a Web page is very heavily loaded with images or other content, your device may not be able to handle the entire page. To reduce the amount of memory Opera will use to display the page, you can set Opera to not download images using Menu > Display > Load images. You may also want to try disabling JavaScript using Menu > Tools > Settings.
Opera displays a "Network problem" message
Ensure that your device has an Internet connection. If it does not, you should set one up now.
