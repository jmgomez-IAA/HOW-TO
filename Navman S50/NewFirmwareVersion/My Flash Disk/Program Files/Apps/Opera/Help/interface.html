Display Modes

Opera offers several display options that can be activated using Menu > Display

Zoom

Select Zoom in and Zoom out to change the zoom of the current page. This option will scale images, text, and other page content. The zoom options are also available from the context menu.

Load images

For faster browsing and lower cost, you may choose to display only images that have already been cached locally by Opera. This setting is changed by selecting Load images from the Display menu.

Full-screen mode

Activate this mode to use the entire screen for browsing. The standard menu and title bars will not be shown in full-screen mode. To return to normal mode, use the Full screen item in the context menu.

Fit to screen

Opera's Small-Screen Rendering (SSR) technology reformats Web pages intended for desktop browsing to fit the screen of your device. This mode is on by default, and is toggled by selecting Fit to screen from the Display menu. If you switch it off, Opera will show Web pages without changing their layout.

Single column

On high-resolution screens, pages may become hard to read in portrait mode. Selecting Menu > Display > Single column forces a simpler rendering mode that normally will increase readability.

Landscape mode

Opera is able to run in either portrait or landscape mode, and will use whichever mode your device is using. Opera can also toggle between landscape and portrait mode when you select Landscape from the Display menu.

Multiple Windows

Opera supports the use of multiple windows. Sometimes, these may be opened by Web pages, but you can also choose to open links in new windows by activating to context menu for the link, and selecting "Open in new window". When a new window is opened, the Tab bar will appear, with a tab for each window. Tap on the tabs to switch to that window, or tap the close button on the Tab bar to close the current window.

Open or close windows by going to Menu > Windows. If more than one browser window is open, switch between them by selecting Next window, or see a list of all open windows by going to Menu > Windows > Windows. The Next window option is also available from the context menu.

Managing Bookmarks

To bookmark a page you are viewing, go to Action > Bookmark this page.

Select Menu > Tools > Manage bookmarks to launch a bookmark editor that lets you organize your bookmarks in folders.

Settings and History

Opera has a number of options and settings you can change according to your preferencees. To access the settings, use Menu > Tools > Settings

The "General" tab allows you to set the home page, say how Opera should identify itself to Web sites, say if Opera should jump directly to the page content when opening a page, block pop-ups, and set the minimum font size that pages are allowed to use. Note: Opera will only jump directly to the page content when "Fit to screen" is enabled, and the page is being displayed on a narrow screen.

If you share or lend your device, you may want to erase your list of visited Web sites, and the "History" tab allows you to clear cache, cookies, and browsing history. There is also an option to specify the size of Opera's cache.

The "Network" tab allows you to define proxy servers, and select which security protocols you want to enable.

The remaining tab allows you to disable JavaScript and image animations, and to set Opera as your default browser. Here you can also specify to Opera the location of your keyboard, if you have one.

Closing Opera

To close Opera, tap the close button, or select Menu &. Normally, when you close Opera, the system will maintain the process so that it starts up where you left off. If the system runs low on resources, it will automatically end the process. You can end the process manually by using the system memory management tool, or you can tell opera to close completely by using Ctrl+Q.
