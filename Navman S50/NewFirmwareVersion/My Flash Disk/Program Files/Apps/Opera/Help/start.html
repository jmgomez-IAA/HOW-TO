Open Web Pages

Tap the address bar to enter or change the page address. Type in an address, and press Return, or tap the "Go" button to the right of the address field, to open the page. You can also use Action > Go to to access the bookmarks folder, and find previously visited addresses in the history folder. To return to Opera's home page, select Action > Go to homepage. "Go to" is also available from the context menu (see below).

To see information about the current page, go to Menu > Tools > View page info.

Navigating on Web Pages

Opera for Pocket PC supports the use of stylus, joystick, or 5-way navigation keys (D-pad) for navigation.

Stylus

Opera uses a stylus in the same way as most other applications for Pocket PC. Tap on buttons, menus, and text inputs to activate them.

Opera also has a context menu that can be activated by tapping and holding the stylus down over the page that is being displayed.

Joystick and 5-way navigation keys

Up, down, left, and right use Opera's spatial navigation to select the next link or form element in that direction. Push follows links and activates form fields.

Filling in forms

When a form input is active, you can use any of the normal input methods to enter information. This includes the keyboard, keypad, IntelliPad, Block Recognizer, Letter Recognizer, Transcriber, or any other input method offered by your device.

Navigating through history

As well as using "Go to" to see previously visited addresses, Opera offers several ways to navigate through history:

  • Tap the "Back" and "Reload" buttons on the address bar.
  • Select Back and Forward from the context menu.
  • Select the "Back", "Forward", or "Reload" option from the Action menu.

Downloads

When you start downloading a file by clicking a link or submitting a form, it will show up in Opera's download manager. Select items in the download manager, and the menus will allow you to stop the download, retransfer it, clear it from the downloads window, delete it, or open it. Select Close or tap the close button to return to the main Opera window.

From the main Opera window, you can open the download manager by going to Menu > Tools > Show downloads.

Downloaded files are saved into My Device\My Documents, and can be accessed using the File Explorer application.

Copy and Paste

Opera supports clipboard functionality. To copy text, select it by tapping and dragging the stylus over it. Then tap and hold to activate the context menu, and select from the available options.
