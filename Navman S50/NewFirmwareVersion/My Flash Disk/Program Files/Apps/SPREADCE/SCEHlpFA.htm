    ABRIR(archivo; [actualizar_vínculos]; [sólo_lectura]; [formato]; [contraseña_protección]; [contraseña_escritura]; [omitir_mensaje]; [origen_archivo]; [separador_personalizado]; [agregar]; [modificable]; [acceso]; [notificar]; [convertidor])

      Simulates the Archivo Abrir menu option.

      archivoThe name of the file to be opened.
      actualizar_vínculosWhether to update external references. The possible values are:
      0Do not update any references
      1Update external references
      2Update remote references (not currently implemented)
      3Update external and remote references (not currently implemented)
      If this parameter is omitted and external references exist then a prompt will be displayed.
      sólo_lecturaWhether to open the file as read-only (not currently implemented).
      formatoThe character used as a delimiter for text files. The possible values are:
      1Tabulación
      2Coma
      3Espacio
      4Punto y coma
      5Nothing
      6Personalizado (specified in the separador_personalizado parameter)
      If this parameter is omitted the text file delimiter will be determined from the file extension.
      contraseña_protecciónThe password required to unprotect a file (not currently implemented).
      contraseña_escrituraThe password required to open a read-only file for writing (not currently implemented).
      omitir_mensajeWhether to suppress the display of the read-only recommended message (not currently implemented).
      origen_archivoThe operating system where the file originated (not currently implemented).
      separador_personalizadoThe character used as a custom delimiter for text files.
      agregarWhether to add the specified file to the current workbook (not currently implemented).
      modificableWhether a template file should be opened in read-write mode (not currently implemented).
      accesoWhether to change the access of a currently open file (not currently implemented).
      notificarWhether to send a message when a shared workbook becomes available (not currently implemented).
      convertidorOverrides the default file type (not currently implemented).

      ABRIR.ARCHIVO([filtro_de_archivo]; [botón]; [título]; [índice_filtro])

        Returns a file name from the Archivo Abrir dialog box, or FALSO if Cancelar is selected.

        filtro_de_archivoOne or more file filters, separated by commas. If this parameter is omitted it defaults to "Todos los archivos (*.*), *.*".
        botónReplacement text for the Open button (not currently implemented).
        títuloReplacement text for the dialog box title. If this parameter is omitted the title is not changed.
        índice_filtroThe index of the initial file filter. If this parameter is omitted it defaults to 1.

        ABRIRA(archivo; [núm_de_acceso])

          Opens a file for reading or writing. The return value is a file number that can be passed as a parameter to the LEERA, LEERALN, ESCRIBIRA, ESCRIBIRALN, POSICIONA TAMAÑOA, and CERRARA functions.

          archivoThe name of the file to open.
          núm_de_accesoThe access required to the file. The possible values are:
          1Open the file for reading and writing. The file must already exist.
          2Open the file for reading only. The file must already exist.
          3Create a new file for reading and writing. The file will be cleared if it already exists.
          If this parameter is omitted it defaults to 1.

          ACTIVAR([ventana_texto]; [panel])

            Specifies the workbook, sheet and/or pane that is to be active.

            ventana_textoThe workbook and/or sheet name. If this parameter is omitted then the active window is not changed.
            panelFor a split window, specifies which pane is to be active. The possible values are:
            1Upper or upper left.
            2Upper right.
            3Lower or lower left.
            4Lower right.
            If this parameter is omitted then the active pane is not changed.

            ACTIVAR.ANTERIOR([libro])

              Simulates the Ctrl+Page Up or Ctrl+Shift+Tab key combination.

              libroThe workbook for which the previous sheet is to be selected. If this parameter is omitted then the previous workbook is selected.

              ACTIVAR.SIGUIENTE([libro])

                Simulates the Ctrl+Page Down or Ctrl+Tab key combination.

                libroThe workbook for which the next sheet is to be selected. If this parameter is omitted then the next workbook is selected.

                  ALERTA(mensaje; [núm_tipo]; [ayuda])

                    Displays a message box containing the specified text. It returns VERDADERO if the OK button is pressed, otherwise FALSO.

                    mensajeThe text to be displayed in the message box.

                    2A box containing an information icon and an OK button.
                    3A box containing an exclamation icon and an OK button.
                    If this parameter is omitted it defaults to 2.
                    ayudaA reference to a Help topic (not currently implemented).

                      ALTO.FILA([alto]; [ref]; [alto_estándar]; [núm_tipo])

                        Simulates the Formato Fila menu options.

                        altoThe new height of the rows. The height is measured in points. This parameter is ignored if alto_estándar is VERDADERO or núm_tipo is specified. If this parameter is omitted the height of the specified rows is not changed.
                        refThe rows whose height is to be changed. If this parameter is omitted it defaults to the current selection on the active worksheet.
                        alto_estándarSpecifies whether the rows are to be set to the standard row height. The possible values are:
                        VERDADEROThe rows are set to the standard height.
                        FALSOThe rows are not set to the standard height.
                        If this parameter is omitted it defaults to FALSO.
                        núm_tipoSpecifies whether the rows are to be hidden, unhidden or automatically sized. This parameter is ignored if alto_estándar is VERDADERO. The possible values are:
                        1The rows are hidden.
                        2The rows are unhidden.
                        3The rows are set to the height of the highest displayed value.
                        If this parameter is omitted the rows will not be hidden, unhidden or auto-sized.
