Gráficos

    To add a chart, use the Insertar Grafico menu option to add a new chart sheet. Then use the Formato Grafico menu option to display the dialog box that contains the following formatting options:

    Tipo de gráfico
    Serie del gráfico
    Título del gráfico
    Ejes del gráfico
    Leyenda del gráfico

    Tipo de gráfico

      The program currently supports the following chart types:

      ColumnasDisplays vertical columns whose heights correspond to the data values.
      BarrasDisplays horizontal bars whose lengths correspond to the data values.
      LineasDisplays a line through points whose vertical positions correspond to the data values, the horizontal positions are evenly spaced.
      CircularDisplays a circle divided up into regions whose relative sizes correspond to the data values.
      AnillosLike the Circular chart but displays one or more series.
      DispersionDisplays a line through points whose x and y coordinates are given by the data values.
      AreasLike the line chart, but the area below the line is filled in, and the series values are stacked.
      RadarDisplays a line through points whose distances from a central point correspond to the data values.

      Serie del gráfico

        This allows you to specify the data values to be used for each series. To add a new series, press the Nuevo... button. To change an existing series, select it in the list box and then press the Edicion... button. To remove an existing series, select it in the list box and press the Eliminar button.

        Note that for Circular charts, only the first series will be displayed.

        When entering or changing a series, you are prompted for the following information:

        DatosNombreAn optional name that is used to identify the series in the list box. It defaults to 'Series1', etc.
        Valores de YA reference to the cells that contain the data to be charted. You can specify more than one reference, separating them with commas, but they must all be from the same sheet. Alternatively you can specify this as an array of values.
        Rótulos de XFor Columnas, Barras, Circular, Lineas and Areas charts, you can specify a reference or array of values that will be used to label the categories. As there is only one set of labels, this must be the same for all the series on one chart. For Dispersion charts, this item is labelled Valores de X and you must enter a reference or array of values to be used as the x co-ordinates of the line.
        DiseñoTipo de líneaAllows you to choose a solid, dashed or dotted line.
        Color de líneaAllows you to specify the color of the line.
        Tipo de marcadorFor Lineas, Dispersion and Radar charts, allows you to choose a marker for the data points.
        Color de marcadorFor Lineas, Dispersion and Radar charts, allows you to choose the foreground and background colors for the marker.
        Color del áreaFor Columnas, Barras, Circular, Anillos and Areas charts, allows you to specify the colour that will be used to fill the interior.
        RótulosTipo del rótuloAllows you to display the x or y values next to the data points, and for Circular and Anillos charts the percentages.

        Título del gráfico

          This allows you to specify the titles that will be used to label the chart itself and the x and y axes. Note that axis titles do not apply to Circular charts.


          Ejes del gráfico

            This allows you to specify the formatting of the x and y axes. The actual options that are available will depend on the currently selected chart type.

            EjeSpecifies whether the axis line will be drawn.
            Líneas De DivisiónSpecifies whether the guidelines will be drawn.
            GraduaciónSpecifies whether the axis tick marks will be drawn.
            ValoresSpecifies whether the axis values will be displayed.
            LogarítmicaSpecifies whether a linear or logarithmic scale is required.
            MínimoSpecifies the minimum value to be used on the scale.
            MáximoSpecifies the maximum value to be used on the scale.
            PasoSpecifies the interval between the values on the scale.

            Leyenda del gráfico

              This allows you to specify whether and where to display the names that identify each series with its line and/or area color. For Circular charts, the information displayed will be the category names (from the 'X labels' entry in the series formatting dialog box), for other chart types it will be the Series names from the chart series list box.
