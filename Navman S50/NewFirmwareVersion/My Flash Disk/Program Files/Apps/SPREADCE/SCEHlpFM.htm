    ABS(número)

      Returns the absolute value of a number.

      númeroThe number for which you want the absolute value.

      ACOS(número)

        Returns the angle in radians corresponding to the arccosine of a number.

        númeroThe number for which you want the arccosine.

        ACOSH(número)

          Returns the angle in radians corresponding to the inverse hyperbolic cosine of a number.

          númeroThe number for which you want the inverse hyperbolic cosine.

          ALEATORIO()

            Returns a random number between 0 and 1.


            ALEATORIO.ENT(mínimo; máximo)

              Returns a random integer in the range you specify.

              mínimoThe lower limit of the range.
              máximoThe upper limit of the range.

              ASENO(número)

                Returns the angle in radians corresponding to the arcsine of a number.

                númeroThe number for which you want the arcsine.

                ASENOH(número)

                  Returns the angle in radians corresponding to the inverse hyperbolic sine of a number.

                  númeroThe number for which you want the inverse hyperbolic sine.

                  ATAN(número)

                    Returns the angle in radians corresponding to the arctangent of a number.

                    númeroThe number for which you want the arctangent.

                    ATAN2(coord_x; coord_y)

                      Returns the angle in radians corresponding to a pair of co-ordinates.

                      coord_xThe x co-ordinate for which you want the angle.
                      coord_yThe y co-ordinate for which you want the angle.

                      ATANH(número)

                        Returns the angle in radians corresponding to the inverse hyperbolic tangent of a number.

                        númeroThe number for which you want the inverse hyperbolic tangent.

                                                                  2More compact
