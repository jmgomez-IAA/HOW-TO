    FALSO()

      Returns the logical value FALSO.


      NO(valor_lógico)

        Returns VERDADERO if the input expression evaluates to FALSO, and vice versa.

        valor_lógicoThe logical expression for which you want the opposite value.

        O(valor_lógico1; [valor_lógico2; ...])

          Returns VERDADERO if any of the input values are VERDADERO, otherwise FALSO.

          valor_lógico1; ...The values that you want to check.

          SI(prueba_lógica; [valor_si_verdadero]; [valor_si_falso])

            Returns one of two other values depending no whether the logical test evaluates to VERDADERO or FALSO.

            prueba_lógicaAn expression that results in VERDADERO or FALSO.
            valor_si_verdaderoThe value to be returned of the logical test is VERDADERO. If this parameter is omitted it defaults to 0.
            valor_si_falsoThe value to be returned of the logical test is FALSO. If this parameter is omitted it defaults to FALSO.

            VERDADERO()

              Returns the logical value VERDADERO.


              Y(valor_lógico1; [valor_lógico2; ...])

                Returns VERDADERO if all of the input values are VERDADERO, otherwise FALSO.

                valor_lógico1; ...The values that you want to check.
