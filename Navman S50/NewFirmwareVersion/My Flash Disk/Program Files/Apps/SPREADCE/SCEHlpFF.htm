    AMORTIZ.LIN(costo; purchase_date; first_period_date; valor_residual; período; tasa; [basis])

      Returns the depreciation for each accounting period.

      costoThe cost.
      purchase_dateThe purchase date.
      first_period_dateThe end date of the first period.
      valor_residualThe salvage value.
      períodoThe period for which you want to calculate the depreciation.
      tasaThe depreciation rate.
      basisThe method used to represent the number of days in the period versus the number of days in a year. The possible values are:
      0US (NASD) 30/360
      1Actual/actual
      2Actual/360
      3Actual/365
      4European 30/360
      If this parameter is omitted it defaults to 0.

                              DVS(costo; valor_residual; vida; período_inicial; período_final; [factor]; [sin_cambios])

                                Returns the depreciation in a specified range of periods using the variable declining balance method.

                                costoThe cost.
                                valor_residualThe salvage value.
                                vidaThe total number of periods.
                                período_inicialThe first period number for which to calculate depreciation.
                                período_finalThe last period number for which to calculate depreciation.
                                factorThe rate at which the balance declines. If this parameter is omitted it defaults to 2.
                                sin_cambiosSpecifies whether to switch to straight-line depreciation when the straight-line depreciation is greater than the declining balance depreciation. The possible values are:
                                FALSOSwitch to straight-line depreciation.
                                VERDADERODo not switch to straight-line depreciation.
                                If this parameter is omitted it defaults to FALSO.
