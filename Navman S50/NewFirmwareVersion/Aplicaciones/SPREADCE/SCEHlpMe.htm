    Menú Archivo

      NuevoOpens a new blank workbook.
      AbrirDisplays the File Open dialog box for you to choose a spreadsheet file to open.
      RecienteDisplays a list of the 9 most recently used (i.e. opened or saved) files as a shortcut for opening them.
      CerrarCloses the current workbook, after prompting you to save any changes.
      GuardarSaves the current workbook back into the file it was opened from. If this spreadsheet was created with the Archivo Nuevo menu option rather than opened with the Archivo Abrir menu option, then you will be asked for a file name to save it under.
      Guardar ComoDisplays the File Save dialog box for you to choose a name and location for saving the current workbook.
      VentanaDisplays a list of the currently open workbooks and allows you to select a different one as the active workbook.
      ImprimirPrints the currently displayed sheet. This option will not be available if your device does not support printing.
      CerrarThis shuts down the program, after prompting you to save any changed workbooks.

      Menú Edición

        DeshacerThis option undoes the last change you made to the spreadsheet, should you make a mistake or just change your mind. There are currently 16 levels of Undo, that is, the program remembers the last 16 things that you did and can step back through them using this menu option.
        CortarThis option removes the contents of the currently selected cells and places them on the clipboard, so that you can paste them somewhere else. Only one selection can be stored on the clipboard at a time, so this will replace whatever was previously on the clipboard.
        CopiarThis option copies the contents of the currently selected cells and places them on the clipboard, so that you can paste them somewhere else. Only one selection can be stored on the clipboard at a time, so this will replace whatever was previously on the clipboard.
        PegarThis option takes the contents of the clipboard and puts it into the currently selected cells.
        Pegado EspecialWhen cells are cut or copied to the clipboard, the program stores not only what is displayed in the cell, but also the formula that is in the cell and the formatting. The Pegar menu option will paste all of this information into the target cells, but this menu option allows you to paste just some of that information. A dialog box is displayed for you to choose what is pasted.
        Rellenar
        Hacia abajoCopies the top row of the selection into the other rows of the selection.
        Hacia la derechaCopies the left column of the selection into the other columns of the selection.
        Hacia arribaCopies the bottom row of the selection into the other rows of the selection.
        Hacia la izquierdaCopies the right column of the selection into the other columns of the selection.
        SerieDisplays a dialog box that allows you to fill the current selection with a series of numbers or dates.
        Borrar
        TodoRemoves the contents and formatting of the currently selected cells.
        FormatoRemoves only the formatting of the currently selected cells, leaves the contents unchanged.
        ContenidoRemoves only the contents of the currently selected cells, leaves the formatting unchanged.
        ComentariosRemoves any comments associated with the currently selected cells.
        EliminarIf entire rows or columns have been selected then this menu option will delete the selected rows or columns. Otherwise (that is, one or more cells have been selected) a dialog box will be displayed containing 4 options: (1) Desplazar hacia la izquierda - delete the selected cells and move the cells on the right of them to the left to take their places, (2) Desplazar hacia arriba - delete the selected cells and move the cells below them up to take their places, (3) Toda la fila - delete the entire rows that the selection spans, the rows below will be moved up, (4) Toda la columna - delete the entire columns that the selection spans, the columns to the right will be moved left.
        Eliminar HojaDelete the currently displayed worksheet, chart or macro sheet. If there is only one worksheet in your spreadsheet file, you will not be allowed to delete it.
        Mover o copiar hojaAllows you to move or copy the current sheet to a different position in the file.
        BuscarAllows you to look for cells that contain specified things. This menu option displays a dialog box asking you 4 things: (1) the text that you want to search for, (2) whether to look for it in the formulas (what was typed into the cells) or in the values (what is displayed in the cells), (3) whether the search is to be case-sensitive (whether you want upper and lower case letters to be treated as different), and (4) whether to find cells that are equal to the text that you typed or just contain the text that you typed. Entering these and pressing Buscar siguiente will move the cursor to the first cell after the current cursor position that matches your request. The dialog box stays on the screen until it is cancelled, so repeated pressing of the Buscar siguiente button will take you to each of the cells that match your request.
        Reemplazar(HPC and .Net versions only) Allows you to search for text in cell formulas and optionally replace one or all occurrences.
        Ir AAllows you to move directly to another part of the worksheet. This menu option displays a dialog box that asks you for a cell reference. Entering the reference and pressing OK will move the cursor to the specified cell.

        Menú Insertar

          CeldasIf entire rows or columns have been selected then this menu option will insert the selected rows or columns. Otherwise (that is, one or more cells have been selected) a dialog box will be displayed containing 4 options: (1) Desplazar hacia la derecha - the cells to the right of the selection will be moved to the right to make room for the new cells, (2) Desplazar hacia abajo - the cells below the selection will be moved down to make room for the new cells, (3) Toda la fila - the entire rows that the selection spans will be inserted, (4) Toda la columna - the entire columns that the selection spans will be inserted.
          FilaInserts one or more new rows where the selection is.
          ColumnaInserts one or more new columns where the selection is.
          HojaInserts a blank worksheet at the end of the spreadsheet file.
          Grafico
          HojaInserts a new chart sheet at the end of the spreadsheet file. If no cells have been selected then the Formato del gráfico dialog box will be displayed, otherwise a default column chart will be built using the selected data.
          ObjetoInserts a new chart object on the current worksheet. Drag the pointer across the worksheet to indicate where the chart should be placed. If no cells have been selected then the Formato del gráfico dialog box will be displayed, otherwise a default column chart will be built using the selected data.
          MacroInserts a blank macro sheet at the end of the spreadsheet file. Macro sheets are used for storing custom (user-defined) functions and commands.
          FuncionesDisplays a dialog box from which you can select a function to be inserted into the current cell formula.
          Nombre
          DefinirDisplays a dialog box showing the names that have been defined for references in this spreadsheet file. It allows you to add new names, change the references of existing names, or to delete names that are no longer required.
          ComentarioThis option allows you to add a comment to the currently selected cell. If there is already a comment attached to the cell, then this option will be changed to Modificar comentario, and the existing comment will be displayed for you to change. You can press the Escape key to leave without entering or changing the comment, or you can click outside the comment box to save the changes. When a cell contains a comment, there will be a comment indicator in the top-right corner of the cell. To display a comment you can either use this menu option, or you can click and hold the pointer on the cell for 1/2 a second, the comment will be displayed until you move or release the pointer. To remove a comment from a cell, use the Editar Borrar Comentarios menu option.
          Control
          Texto estáticoInserts a label.
          Cuadro de grupoInserts a group box.
          BotónInserts a command button.
          Casilla de verificaciónInserts a check box.
          Botón de opciónInserts an option button.
          Cuadro de listaInserts a list box.
          Lista desplegableInserts a combo box.
          Barra de desplazamientoInserts a horizontal or vertical scroll bar.
          Control de giroInserts a spin control.
          Objeto de dibujo
          LíneaInserts a line.
          RectánguloInserts a rectangle.
          ElipseInserts an oval.
          Cuadro de textoInserts a text box.
          AutoformasInserts lines, connectors and other shapes.
          HipervínculoDisplays a dialog box that allows you to add, change or remove a link from the current selection to a file and/or location. To specify a location in the current file leave the file name blank. To remove a link, delete the file name and the location name.

          Menú Formato

            CeldasDisplays a dialog box that allows you to change the way that the contents of the selected cells are displayed. You can change the way that numbers, amounts of money, dates, times and text are formatted. You can change the colour of the text, the pattern and colour of the cell background, and the type and colour of the cell borders.
            Fila
            AltoDisplays a dialog box showing the current height of the selected rows in points (the unit of measurement used for text), and allows you to change it. You can also change the row height by dragging the separator between the row numbers.
            AutoajustarSets the row height to the largest required for the values displayed in the row. You can also autofit a row by double-clicking on the separator below the row number.
            OcultarHides the selected rows.
            MostrarUnhides the selected rows. Because you cannot select individual hidden rows, you will have to use the Editar>Ir A menu option to move the cursor to the required row, or select all the rows from the one above the hidden rows to the one below, before using this option.
            Columna
            AnchoDisplays a dialog box showing the current width of the selected columns in characters, and allows you to change it. You can also change the column width by dragging the separator between the column headings.
            Ajustar a la selecciónSets the column width to the largest required for the values displayed in the selected cells. You can also autofit a column by double-clicking on the separator to the right of the column heading.
            OcultarHides the selected columns.
            MostrarUnhides the selected columns. Because you cannot select individual hidden columns, you will have to use the Editar>Ir A menu option to move the cursor to the required column, or select all the columns from the one to the left of the hidden columns to the one to the right, before using this option.
            Ancho EstándarDisplays a dialog box showing the default width of the columns on the current sheet, and allows you to change it.
            Hoja
            Cambiar NombreDisplays a dialog box showing the current sheet name, and allows you to change it.
            OcultarAllows you to hide the currently displayed sheet.
            MostrarAllows you to select a hidden sheet to be displayed.
            Líneas De DivisiónAllows you to turn the gridlines on and off for the current sheet.
            TítulosAllows you to turn the row and column headings on and off for the current sheet.
            ZoomAllows you to set the zoom level for the current sheet.
            ProtegerAllows you to turn protection on and off for the current sheet. When protection is on, only cells that are not locked (set by using the Formato>Celdas menu option) can be changed.
            FórmulasAllows you to toggle the display of the cell formulas and cell values for the current sheet.
            Color de etiquetaAllows you to change the tab color for the current sheet.
            Archivo
            Barra de desplazamiento verticalAllows you to select whether the Vertical scrollbar is to be displayed for sheets in the current file.
            Barra de desplazamiento horizontalAllows you to select whether the Horizontal scrollbar is to be displayed for sheets in the current file.
            Etiquetas de la hojasAllows you to select whether the Sheet tabs are to be displayed for sheets in the current file.
            Programa
            Barras de herramientasAllows you to show or hide the tool bars. The tool bar buttons provide short cuts to commonly-used menu options.
            Barra de fórmulasAllows you to show or hide the formula bar. This is the area where formulas are entered and edited.
            Barra de estadoAllows you to show or hide the status bar. The status bar shows the current cursor position, and has a menu (activated by double-click) that allows you to select items to be displayed in the status bar, such as the sum of the selected cells.
            Pantalla completaAllows you to show or hide the toolbars, formula bar, status bar, headings, scroll bars and sheet tabs so that more cells can be displayed.
            CondicionalAllows you to select alternative formatting for cells depending on conditions.
            InmovilizarOn a worksheet, this allows you to fix the currently displayed row and/or column headings so that they will not move when the sheet is scrolled. When you use this option, the rows above the cursor cell and the columns to the left of it will be fixed. For example, if you want the top row to stay the same when you page down through the sheet, move the cursor to cell A2 and then use this menu option, the row above cell A2, which is row 1, will stay displayed when you page through the sheet, and as there are no columns to the left of cell A2, no columns will be fixed. When this option has been used, the menu option changes to 'Unfreeze' which allows you to free the rows and columns again.
            Datos
            OrdenarDisplays a dialog box that allows you to sort the currently selected cells, either by rows or columns, in ascending or descending order.
            Filtro
            AutoFiltroAllows you to filter the spreadsheet rows by specifying selection criteria.
            Mostrar todoTurns off any active filters.
            Filtro avanzadoAllows you to filter the spreadsheet rows by specifying more complex criteria.
            FichaAllows you to display, update and search a table of data one record at a time.
            ValidaciónAllows you to restrict the range of values that can be entered into a cell, or display a drop-down list to choose a value from, and display input and error messages.
            Buscar objetivoAllows you to set a cell to a specific value by changing another cell.
            GraficoOn a chart sheet, this displays a dialog box that allows you to change the type of the chart, the data that is used, and the way in which the chart is formatted.
            ControlDisplays a dialog box that allows you to set the attributes of the currently selected Control object.
            Objeto de dibujoDisplays a dialog box that allows you to set the attributes of the currently selected Drawing object.
