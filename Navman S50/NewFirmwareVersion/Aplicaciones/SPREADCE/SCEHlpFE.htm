            BIN.A.DEC(número)

              Returns the decimal equivalent of a binary number.

              númeroThe binary number to be converted. It can be a number or a string up to 10 (binary) digits long.

              BIN.A.HEX(número; [posiciones])

                Returns a string that is the hexadecimal equivalent of a binary number.

                númeroThe binary number to be converted. It can be a number or a string up to 10 (binary) digits long.
                posicionesThe minimum length required in the returned string. If the string is less than this length then it will be padded with leading zeros. If this parameter is omitted it defaults to zero.

                BIN.A.OCT(número; [posiciones])

                  Returns a string that is the octal equivalent of a binary number.

                  númeroThe binary number to be converted. It can be a number or a string up to 10 (binary) digits long.
                  posicionesThe minimum length required in the returned string. If the string is less than this length then it will be padded with leading zeros. If this parameter is omitted it defaults to zero.

                    imag_numThe imaginary part of the complex number.
                    sufijoThe suffix to be used for the imaginary part. Should be "i" or "j". If this parameter is omitted it defaults to "i".

                      DEC.A.BIN(número; [posiciones])

                        Returns a string that is the binary equivalent of a decimal number.

                        númeroThe decimal number to be converted.
                        posicionesThe minimum length required in the returned string. If the string is less than this length then it will be padded with leading zeros. If this parameter is omitted it defaults to zero.

                        DEC.A.HEX(número; [posiciones])
