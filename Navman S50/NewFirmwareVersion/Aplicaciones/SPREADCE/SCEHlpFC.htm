Control de macros funciones

    [ ] indicates optional parameters

    ARGUMENTO([nombre_texto]; [tipo_de_dato]; [ref])
    DETENER([cancelar_cerrar])
    ESPERAR([núm_de_serie])
    ESTABLECER.NOMBRE(nombre_texto; [valor])
    ESTABLECER.VALOR(ref; valores)
    FIN.SI()
    INTERRUMPIR()
    IR.A(ref)
    MIENTRAS(prueba_lógica)
    PARA(contador; núm_inicial; núm_final; [incremento])
    PARA.CELDA(nombre_ref; [ref_área]; [saltar_blancos])
    REINICIAR([núm_de_nivel])
    RETORNO([valor])
    SALIR.BUCLE()
    SI(prueba_lógica)
    SI.NO()
    SI.NO.SI(prueba_lógica)

    ARGUMENTO([nombre_texto]; [tipo_de_dato]; [ref])

      Defines an argument for a custom function.

      nombre_textoThe defined name that will be assigned to the argument. If this parameter is omitted then no name will be assigned.
      tipo_de_datoThe acceptable data type(s) for the argument value. It can be any combination of the following (to specify more than one value, add the numbers together):
      1Número
      2Texto
      4Lógico
      8Referencia
      16Error
      64Matriz
      If this parameter is omitted it defaults to 7.
      refThe cell reference where the argument value will be stored. If this parameter is omitted then the argument value will not be stored.

      DETENER([cancelar_cerrar])

        Terminates all running macros.

        cancelar_cerrarWhether, in an Auto_cerrar macro, to prevent the workbook from being closed (not currently implemented).

        ESPERAR([núm_de_serie])

          Forces the macro to wait until the date/time specified by the serial number.

          núm_de_serieThe date/time when the macro is to resume. If this parameter is omitted the macro does not wait.

          ESTABLECER.NOMBRE(nombre_texto; [valor])

            Assigns a defined name to the specified value.

            nombre_textoThe defined name to use.
            valorThe value to be associated with the name. If this parameter is omitted then the name is deleted.

            ESTABLECER.VALOR(ref; valores)

              Stores the specified values in the specified cells on the macro sheet. Do not use this function to try to update cells on a worksheet.

              refThe cells where the values are to be stored.
              valoresThe values to be stored.

              FIN.SI()

                Specifies the end of the block of statements conditioned by the corresponding SI or ELSE.IF.


                INTERRUMPIR()

                  Ends the processing of a PARA, PARA.CELDA or MIENTRAS loop. The macro will continue with the statement after the SALIR.BUCLE function.


                  IR.A(ref)

                    Forces the macro to continue with the statement at the reference.

                    refThe reference of the cell where macro execution is to continue.

                    MIENTRAS(prueba_lógica)

                      Specifies a logical test that conditions the execution of a block of statements multiple times. The statements between the MIENTRAS and SALIR.BUCLE functions will be repeated as long as the logical test returns the value VERDADERO.

                      prueba_lógicaAn expression that results in VERDADERO or FALSO.

                      PARA(contador; núm_inicial; núm_final; [incremento])

                        Performs a group of instructions a specified number of times. The end of the group is indicated by a SALIR.BUCLE function.

                        contadorA defined name that will be used to store the current index.
                        núm_inicialThe initial value to be assigned to the counter.
                        núm_finalThe last value to be assigned to the counter.
                        incrementoThe value to be added to the counter each time around the loop. If this parameter is omitted it defaults to 1.

                        PARA.CELDA(nombre_ref; [ref_área]; [saltar_blancos])

                          Performs a group of instructions for each cell in a specified area. The end of the group is indicated by a SALIR.BUCLE function.

                          nombre_refA defined name that will be used to store the current reference.
                          ref_áreaThe range of cells to be processed.
                          If this parameter is omitted it defaults to the current selection.
                          saltar_blancosWhether blank cells are to be skipped. The possible values are:
                          FALSOAll cells in the area are processed
                          VERDADEROOnly the non-blank cells in the area are processed
                          If this parameter is omitted it defaults to FALSO.

                          REINICIAR([núm_de_nivel])

                            Specifies the number of levels that will be skipped when this macro returns.

                            núm_de_nivelThe number of levels to skip. If this parameter is omitted then all levels are skipped.

                            RETORNO([valor])

                              Forces the macro to end, and if it is a custom function then to return the specified value.

                              valorFor custom functions, the value to be returned.

                              SALIR.BUCLE()

                                Specifies the end of the block of statements contained in a PARA, PARA.CELDA or MIENTRAS loop.


                                SI(prueba_lógica)

                                  Specifies a logical test that conditions the execution of a block of statements.

                                  prueba_lógicaAn expression that results in VERDADERO or FALSO.

                                  SI.NO()

                                    Introduces the block of statements to be processed when the corresponding SI or ELSE.IF condition is false.


                                    SI.NO.SI(prueba_lógica)

                                      Specifies another logical test that conditions a block of statements when the corresponding SI or ELSE.IF condition is false.

                                      prueba_lógicaAn expression that results in VERDADERO or FALSO.
