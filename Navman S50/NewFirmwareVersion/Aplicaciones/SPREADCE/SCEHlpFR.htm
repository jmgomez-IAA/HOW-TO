    AREAS(ref)

      Returns the number of areas contained in the reference.

      refThe reference whose areas you want to count.

      BUSCAR(valor_buscado; vector_de_comparación; [vector_resultado])

        Returns a value from a horizontal or vertical table, found by searching for the lookup value in the top row (for a horizontal table) or left column (for a vertical or square table) of the table and then returning a value from the corresponding position in the result range (if specified) or from the bottom row (for a horizontal table) or right column (for a vertical or square table) of the table.

        valor_buscadoThe value to be found in the table.
        vector_de_comparaciónA reference containing the table cells.
        vector_resultadoThe range of cells from which to return a result. If this parameter is not specified the result will be returned from the opposite row or column of the table range.

        BUSCARH(valor_buscado; matriz_buscar_en; indicador_filas; [ordenado])

          Returns a value from a horizontal table, found by searching for the lookup value in the top row of the table and then returning a value from the same or a different row in the table.

          valor_buscadoThe value to be found in the table.
          matriz_buscar_enA reference containing the table cells.
          indicador_filasThe offset of the value to be returned, where 1 is the top row of the table.
          ordenadoWhether to find an approximate or exact match. The possible values are:
          FALSOAn exact match is required
          VERDADEROIf an exact match is not found, use the closest value less than the lookup value
          If this parameter is omitted it defaults to VERDADERO.

          BUSCARV(valor_buscado; matriz_buscar_en; indicador_columnas; [ordenado])

            Returns a value from a vertical table, found by searching for the lookup value in the left column of the table and then returning a value from the same or a different column in the table.

            valor_buscadoThe value to be found in the table.
            matriz_buscar_enA reference containing the table cells.
            indicador_columnasThe offset of the value to be returned, where 1 is the left column of the table.
            ordenadoWhether to find an approximate or exact match. The possible values are:
            FALSOAn exact match is required
            VERDADEROIf an exact match is not found, use the closest value less than the lookup value
            If this parameter is omitted it defaults to VERDADERO.

            COINCIDIR(valor_buscado; matriz_buscada; [tipo_de_coincidencia])

              Returns a number representing the position of a value in a table.

              valor_buscadoThe value to be found in the table.
              matriz_buscadaA reference containing the table cells.
              tipo_de_coincidenciaWhether to find an approximate or exact match. The possible values are:
              1If an exact match is not found, use the closest value less than the lookup value (the table values should be in ascending order)
              0An exact match is required
              -1If an exact match is not found, use the closest value greater than the lookup value (the table values should be in descending order)
              If this parameter is omitted it defaults to 1.

              COLUMNA([ref])

                Returns the column number of the reference.

                refThe reference whose column number you want. If this parameter is omitted it defaults to the cell containing the function.

                COLUMNAS(rango)

                  Returns the number of columns in the reference.

                  rangoThe reference whose columns you want to count.

                  DESREF(ref; filas; columnas; [alto]; [ancho])

                    Returns a new reference based on the specified reference.

                    refThe reference to be used as a starting point.
                    filasThe number of rows to move the reference up (negative) or down (positive).
                    columnasThe number of columns to move the reference left (negative) or right (positive).
                    altoThe height of the new reference. If this parameter is omitted it defaults to the height of the old reference.
                    anchoThe width of the new reference. If this parameter is omitted it defaults to the width of the old reference.

                    DIRECCION(fila; columna; [abs]; [a1]; [hoja])

                      Returns a string containing the specified cell address.

                      filaThe row number of the cell.
                      columnaThe column number of the cell.
                      absA number representing whether the row or column are to be absolute or relative. The possible values are:
                      1Both absolute
                      2Absolute row, relative column
                      3Relative row, absolute column
                      4Both relative
                      If this parameter is omitted it defaults to 1.
                      a1Specifies the style of the reference. The possible values are:
                      FALSOR1C1 style
                      VERDADEROA1 style
                      If this parameter is omitted it defaults to VERDADERO.
                      hojaOptional sheet name with which to prefix the reference.

                      ELEGIR(núm_índice; valor1; [valor2; ...])

                        Returns one of several values depending on the index.

                        núm_índiceThe index of the value to be returned, should be in the range 1 to 29.
                        valor1; ...Up to 29 values, one of which will be chosen to be the result.

                        FILA([ref])

                          Returns the row number of the reference.

                          refThe reference whose row number you want. If this parameter is omitted it defaults to the cell containing the function.

                          FILAS(rango)

                            Returns the number of rows in the reference.

                            rangoThe reference whose rows you want to count.

                            HIPERVINCULO(ubicación_del_vínculo; [nombre_descriptivo])

                              Jumps to a cell or range when this cell is selected.

                              ubicación_del_vínculoA text expression that evaluates to the form "filename" or "[filename]reference".
                              nombre_descriptivoThe text to be displayed in the cell. If this parameter is omitted it defaults to the link location text.

                              INDICE(matriz; [núm_fila]; [núm_columna]; [núm_área])

                                Returns a subset of an array or reference.

                                matrizThe array or reference of which you want the subset.
                                núm_filaThe number of the row to return. If this parameter is omitted all rows will be returned.
                                núm_columnaThe number of the column to return. If this parameter is omitted all columns will be returned.
                                núm_áreaThe number of the area to return when the reference contains more than one area. If this parameter is omitted it defaults to 1.

                                INDIRECTO(ref; [a1])

                                  Returns a reference from the specified text.

                                  refA text expression that evaluates to the name of a cell or range of cells.
                                  a1Specifies the style of the reference. The possible values are:
                                  FALSOR1C1 style
                                  VERDADEROA1 style
                                  If this parameter is omitted it defaults to VERDADERO.

                                  TRANSPONER(rango)

                                    Returns the transposition of the specified array or reference.

                                    rangoThe array or reference whose values you want to transpose.
