    CELDA(tipo_de_info; [ref])

      Returns information about a worksheet cell.

      tipo_de_infoThe type of information you want returned. The possible values are:
      "ANCHO"The cell width in characters
      "COLOR"1 if the cell is formatted in color for negative values, otherwise 0 (not currently implemented)
      "COLUMNA"The column number of the cell
      "CONTENIDO"The result of the formula in the cell, or 0 if the cell is empty
      "DIRECCION"A string containing the cell reference
      "FILA"The row number of the cell
      "FORMATO"The number format that has been applied to the cell
      "NOMBREARCHIVO"The path, filename and sheet name of the cell
      "PARENTESIS"1 if the cell is formatted in parentheses for positive values, otherwise 0 (not currently implemented)
      "PREFIJO"The alignment of text in the cell:
      'left-aligned
      "right-aligned
      ^centred
      \filled
      "PROTEGER"1 if the cell is locked, otherwise 0
      "TIPO"A string representing the type of the cell result: "b" for blank, "l" for label (string), otherwise "v"
      refThe cell about which you want information. If this parameter is omitted it defaults to the cell containing the formula.

      CONTAR.BLANCO(rango)

        Returns the number of blank cells in a range.

        rangoThe reference of the cells you want to check.

        ES.IMPAR(número)

          Returns VERDADERO if the number is odd, otherwise FALSO.

          númeroThe number you want to check.

          ES.PAR(número)

            Returns VERDADERO if the number is even, otherwise FALSO.

            númeroThe number you want to check.

            ESBLANCO(valor)

              Returns VERDADERO if the value is a reference to an empty cell, otherwise FALSO.

              valorThe cell whose value you want to check.

              ESERR(valor)

                Returns VERDADERO if the value is an error other than #N/A, otherwise FALSO.

                valorThe expression whose result you want to check.

                ESERROR(valor)

                  Returns VERDADERO if the value is an error, otherwise FALSO.

                  valorThe expression whose result you want to check.

                  ESLOGICO(valor)

                    Returns VERDADERO if the value is a logical value, otherwise FALSO.

                    valorThe expression whose result you want to check.

                    ESNOD(valor)

                      Returns VERDADERO if the value is the error #N/A, otherwise FALSO.

                      valorThe expression whose result you want to check.

                      ESNOTEXTO(valor)

                        Returns VERDADERO if the value is not a string, otherwise FALSO.

                        valorThe expression whose result you want to check.

                        ESNUMERO(valor)

                          Returns VERDADERO if the value is a number, otherwise FALSO.

                          valorThe expression whose result you want to check.

                          ESREF(valor)

                            Returns VERDADERO if the value is a reference, otherwise FALSO.

                            valorThe expression whose result you want to check.

                            ESTEXTO(valor)

                              Returns VERDADERO if the value is a string, otherwise FALSO.

                              valorThe expression whose result you want to check.

                              INFO(tipo)

                                Returns information about the current spreadsheet.

                                tipoThe type of information you want returned. The possible values are:
                                "ARCHIVOS"The number of spreadsheet files that are open
                                "DIRECTORIO"The path of the current directory
                                "MEMDISP"The number of bytes of memory available
                                "MEMTOT"The total number of bytes of memory
                                "MEMUSADA"The number of bytes of memory used
                                "ORIGEN"A string containing the address of the cell in the top-left corner of the scrollable region
                                "RECALC"A string containing "Automatic" or "Manual", depending on the recalculation mode
                                "SISTEMA"The operating system type
                                "VERSION"The release version with which this program is compatible
                                "VERSIONSO"The operating system version number

                                N(valor)

                                  Returns a number corresponding to the input value, or 0 if the input value is a string. You should not need to use this function, as values are automatically converted where necessary when they are used in formulas.

                                  valorThe value you want as a number.

                                  NOD()

                                    Returns the error value #N/A.


                                    TIPO(valor)

                                      Returns a number corresponding to the type of the value.

                                      valorThe value, or cell containing the value, that you want to check.
                                      The returned values are:
                                      número1
                                      texto2
                                      lógico4
                                      error16
                                      matriz64

                                      TIPO.DE.ERROR(valor_de_error)

                                        Returns a number corresponding to the type of the error.

                                        valor_de_errorThe error value, or cell containing the error value, that you want to check.

                                        #¡NULO!1
                                        #¡DIV/0!2
                                        #¡VALOR!3
                                        #¡REF!4
                                        #¿NOMBRE?5
                                        #¡NUM!6
                                        #N/A7
