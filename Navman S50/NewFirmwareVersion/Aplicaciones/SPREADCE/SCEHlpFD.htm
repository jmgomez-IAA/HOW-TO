    BDCONTAR(base_de_datos, [nombre_de_campo], criterios)

      Returns the count of the number of rows in the database that meet the selection criteria and where the specified field contains a number.

      base_de_datosA reference to the cells that contain the database (including the column headings). It is usually easiest to use a defined name for this.
      nombre_de_campoThe column number or column heading name of the values that you want to count. If this parameter is omitted then the function will return the number of database rows that meet the selection criteria.
      criteriosA reference to cells that contain the criteria that you want to use when selecting records from the database to process. You can specify the same range as used for the base_de_datos parameter if you want to select all the rows.

            BDEXTRAER(base_de_datos, nombre_de_campo, criterios)

              Returns the value of a specified field from the row in the database range that meets the selection criteria. There should be one and only one row that meets the selection criteria. If no rows meet the criteria then the error #¡VALOR! is returned. If more than one row meets the criteria then the error #¡NUM! is returned.

              base_de_datosA reference to the cells that contain the database (including the column headings). It is usually easiest to use a defined name for this.
              nombre_de_campoThe column number or column heading name of the value that you want to retrieve.
              criteriosA reference to cells that contain the criteria that you want to use when selecting records from the database to process. You can specify the same range as used for the base_de_datos parameter if you want to select all the rows.
