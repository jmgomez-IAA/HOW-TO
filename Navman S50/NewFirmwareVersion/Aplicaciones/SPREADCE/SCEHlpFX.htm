    CARACTER(número)

      Returns the character corresponding to the specified position in the character set.

      númeroThe number of the character that you want.

      CODIGO(texto)

        Returns the position in the character set of the first character in the string.

        textoThe character of which you want the character code.

        CONCATENAR(texto1; [texto2; ...])

          Returns a string consisting of the input strings concatenated together.

          texto1; ...The strings that you want to concatenate.

          DECIMAL(número; [decimales]; [no_separar_millares])

            Returns a string consisting of the input number rounded to the specified number of decimal places.

            númeroThe number that you want to format.
            decimalesThe number of decimal places required. If this parameter is omitted it defaults to 2.
            no_separar_millaresSpecifies whether to suppress the thousands separator in the formatted number. The possible values are:
            FALSOThe thousands separator is used.
            VERDADEROThe thousands separator is not used.
            If this parameter is omitted it defaults to FALSO.

            DERECHA(texto; [núm_de_caracteres])

              Returns a string consisting of the specified number of characters from the end of the input string.

              textoThe string of which you want the end.
              núm_de_caracteresThe number of characters to extract from the string. If this parameter is omitted it defaults to 1.

              ENCONTRAR(texto_buscado; dentro_del_texto; [núm_inicial])

                Returns a number corresponding to the position of one text string within another (case-sensitive).

                texto_buscadoThe text you want to find.
                dentro_del_textoThe text to be searched.
                núm_inicialThe position at which to start searching. If this parameter is omitted it defaults to 1.

                ESPACIOS(texto)

                  Returns a string consisting of the input string with any extra spaces removed.

                  textoThe string that you want to format.

                  EXTRAE(texto; posición_inicial; núm_de_caracteres)

                    Returns a string consisting of the specified number of characters from the specified position of the input string.

                    textoThe string of which you want the extract.
                    posición_inicialThe starting position of characters to extract from the string.
                    núm_de_caracteresThe number of characters to extract from the string.

                    HALLAR(texto_buscado; dentro_del_texto; [núm_inicial])

                      Returns a number corresponding to the position of one text string within another (not case-sensitive).

                      texto_buscadoThe text you want to find.
                      dentro_del_textoThe text to be searched.
                      núm_inicialThe position at which to start searching. If this parameter is omitted it defaults to 1.

                      IGUAL(texto1; texto2)

                        Returns VERDADERO if the input strings are exactly the same, otherwise FALSO.

                        texto1The first string to be compared.
                        texto2The second string to be compared.

                        IZQUIERDA(texto; [núm_de_caracteres])

                          Returns a string consisting of the specified number of characters from the beginning of the input string.

                          textoThe string of which you want the beginning.
                          núm_de_caracteresThe number of characters to extract from the string. If this parameter is omitted it defaults to 1.

                          LARGO(texto)

                            Returns the length of the input string.

                            textoThe string of which you want the length.

                            LIMPIAR(texto)

                              Returns a string corresponding to the input string with any unprintable characters removed.

                              textoThe text from which you want to remove unprintable characters.

                              MAYUSC(texto)

                                Returns a string which is the input string with all of the characters converted to upper case.

                                textoThe string that you want to convert.

                                MINUSC(texto)

                                  Returns a string which is the input string with all of the characters converted to lower case.

                                  textoThe string you want to convert.

                                  MONEDA(número; [decimales])

                                    Returns a string consisting of the input number rounded to the specified number of decimal places and converted to currency format.

                                    númeroThe amount that you want to format.
                                    decimalesThe number of decimal places required. If this parameter is omitted it defaults to 2.

                                    NOMPROPIO(texto)

                                      Returns a string which is the input string with the first letter of each word converted to upper case and all of the other characters converted to lower case.

                                      textoThe string you want to convert.

                                      REEMPLAZAR(texto_original; núm_inicial; núm_de_caracteres; texto_nuevo)

                                        Returns a string which is the input string with a specified portion replaced.

                                        texto_originalThe string you want to convert.
                                        núm_inicialThe starting position of the text to be removed.
                                        núm_de_caracteresThe number of characters to be removed.
                                        texto_nuevoThe text to be inserted into the string.

                                        REPETIR(texto; núm_de_veces)

                                          Returns a string which is the input string repeated a specified number of times.

                                          textoThe string you want to repeat.
                                          núm_de_vecesThe number of times to repeat the string.

                                          SUSTITUIR(texto; texto_original; texto_nuevo; [núm_de_ocurrencia])

                                            Returns a string which is the input string with specified text replaced.

                                            textoThe string you want to convert.
                                            texto_originalThe string to be removed from the original text.
                                            texto_nuevoThe string to be inserted in the original text.
                                            núm_de_ocurrenciaSpecifies which occurrence of old_text is to be replaced. If this parameter is omitted then all occurrences are replaced.

                                            T(valor)

                                              Returns a string which is the input value if it is text, otherwise an empty string.

                                              valorThe value you want if it is a string.

                                              TEXTO(valor; formato)

                                                Returns a string consisting of the input number formatted using the specified format string.

                                                valorThe number that you want to format.
                                                formatoThe format string to use.

                                                VALOR(texto)

                                                  Returns a number which corresponds to the value in the input string. You should not need to use this function, as values are automatically converted from text to numeric when required in formulas.

                                                  textoThe string that you want to convert to a number.
