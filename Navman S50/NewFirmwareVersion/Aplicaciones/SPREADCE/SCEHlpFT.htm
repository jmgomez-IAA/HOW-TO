    AHORA()

      Returns the serial number corresponding to the current date and time.


      AÑO(núm_de_serie)

        Returns the year number (greater than or equal to 1900) from the specified date/time value.

        núm_de_serieThe date/time value from which you want to extract the year number.

        DIA(núm_de_serie)

          Returns the day number (between 1 and 31) from the specified date value.

          núm_de_serieThe date value from which you want to extract the day number.

          DIA.LAB(fecha_inicial; días; [días_de_fiesta])

            Returns the working day that is the start date plus or minus a number of working days (weekdays), excluding any holidays.

            fecha_inicialThe 'from' date.
            díasThe number of working days to add or subtract.
            días_de_fiestaA date or array of dates or a reference to a list of dates that are to be treated as non-working days.

            DIAS.LAB(fecha_inicial; fecha_final; [días_de_fiesta])

              Returns the number of working days (weekdays) between the start and end dates, excluding any holidays.

              fecha_inicialThe 'from' date.
              fecha_finalThe 'to' date.
              días_de_fiestaA date or array of dates or a reference to a list of dates that are to be treated as non-working days.

              DIAS360(fecha_inicial; fecha_final; [método])

                Returns the number of days between two dates using a 360-day year (30 day months).

                fecha_inicialThe 'from' date.
                fecha_finalThe 'to' date.
                métodoIndicates how to deal with end dates that are the 31st of the month (the start date will always be moved from the 31st to the 30th). The options are:
                FALSOUS (NASD) method (if start date is less than the 30th then set end date to 1st of next month, else set end date to 30th).
                VERDADEROEuropean method (end date moved form 31st to 30th).
                If this parameter is omitted it defaults to FALSO.

                DIASEM(núm_de_serie; [tipo])

                  Returns a number representing the day of the week of a given date.

                  núm_de_serieThe date/time value from which you want the day of the week.
                  tipoThe range of values returned and what they represent. The possible values are:
                  1From 1 to 7, where 1 = Domingo and 7 = Sábado.
                  2From 1 to 7, where 1 = Lunes and 7 = Domingo.
                  3From 0 to 6, where 0 = Lunes and 6 = Domingo.
                  If this parameter is omitted it defaults to 1.

                  FECHA(año; mes; día)

                    Returns the date value corresponding to a specified year, month and day. Note that only dates from 1900 onwards can be used.

                    añoThe year number. If you enter a number less than 1900 then 1900 will be added to the value.
                    mesThe month number. Preferably between 1 and 12.
                    díaThe day number. Preferably between 1 and 31.

                    FECHA.MES(fecha_inicial; meses)

                      Returns the date that is the start date plus or minus a number of months.

                      fecha_inicialThe 'from' date.
                      mesesThe number of months to be added or subtracted.

                      FECHANUMERO(texto_de_fecha)

                        Returns the date value that corresponds to a date in text form. Note that only dates from 1900 onwards can be used. You should not need to use this function, as dates are automatically converted from text to date values when they are used in formulas.

                        texto_de_fechaThe date that you want to convert. It can contain date separators and/or a month name. If only the day and month is supplied then the year will default to the current year.

                        FIN.MES(fecha_inicial; meses)

                          Returns the date that is the last day of the month calculated from the start date plus or minus a number of months.

                          fecha_inicialThe 'from' date.
                          mesesThe number of months to be added or subtracted.

                          FRAC.AÑO(fecha_inicial; fecha_final; [basis])

                            Returns the fraction of a year represented by the difference between two dates.

                            fecha_inicialThe 'from' date.
                            fecha_finalThe 'to' date.
                            basisThe method used to represent the number of days in the period versus the number of days in a year. The possible values are:
                            0US (NASD) 30/360
                            1Actual/actual
                            2Actual/360
                            3Actual/365
                            4European 30/360
                            If this parameter is omitted it defaults to 0.

                            HORA(núm_de_serie)

                              Returns the hour number (between 0 and 23) from the specified date/time value.

                              núm_de_serieThe date/time value from which you want to extract the hour number.

                              HORANUMERO(texto_de_hora)

                                Returns the time value that corresponds to a time in text form. You should not need to use this function, as times are automatically converted from text to time values when they are used in formulas.

                                texto_de_horaThe time that you want to convert. It can contain time separators and/or AM/PM designators. If seconds or minutes and seconds are not specified then they will default to zero.

                                HOY()

                                  Returns the serial number corresponding to the current date.


                                  MES(núm_de_serie)

                                    Returns the month number (between 1 and 12) from the specified date/time value.

                                    núm_de_serieThe date/time value from which you want to extract the month number.

                                    MINUTO(núm_de_serie)

                                      Returns the minute number (between 0 and 59) from the specified date/time value.

                                      núm_de_serieThe date/time value from which you want to extract the minute number.

                                      NSHORA(hora; minuto; segundo)

                                        Returns the serial number corresponding to a specified hour, minute and second.

                                        horaThe hour number. Preferably between 0 and 23.
                                        minutoThe minute number. Preferably between 0 and 59.
                                        segundoThe second number. Preferably between 0 and 59.

                                        NUM.DE.SEMANA(núm_de_serie; [tipo])

                                          Returns the week of the year of a given date.

                                          núm_de_serieThe date/time value from which you want the week number.
                                          tipoThe method used to calculate the week number. The possible values are:
                                          0International Standard (ISO) week numbers. Weeks begin on Lunes. Week 1 is the first week where 4 or more days are in the specified year.
                                          1Microsoft week numbers. Weeks begin on Domingo. Week 1 is the week that contains January 1st.
                                          2Microsoft week numbers. Weeks begin on Lunes. Week 1 is the week that contains January 1st.
                                          If this parameter is omitted it defaults to 1.

                                          SEGUNDO(núm_de_serie)

                                            Returns the second number (between 0 and 59) from the specified date/time value.

                                            núm_de_serieThe date/time value from which you want to extract the second number.

                                            SIFECHA(fecha1; fecha2; units)

                                              Returns the difference between two dates in the units that you specify.

                                              fecha1The 'from' date.
                                              fecha2The 'to' date. This should be greater than or equal to the 'from' date.
                                              unitsThe units in which you want the difference expressed. The possible values are:
                                              "y"The difference in years.
                                              "m"The difference in months.
                                              "d"The difference in days.
                                              "ym"The difference in months, dropping any whole years (the result will always be between 0 and 11).
                                              "yd"The difference in days, dropping any whole years (the result will always be between 0 and 365).
                                              "md"The difference in days, dropping any whole months (the result will always be between 0 and 30).
