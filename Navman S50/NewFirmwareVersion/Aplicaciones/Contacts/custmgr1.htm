Getting Started ...
  • To add a new customer, select Clear from Edit menu, type in customer ID, and hit the button near the field.
  • To save a customer record, select Save from Edit menu.
  • To delete a customer record, select Delete from Edit menu.
  • To view different customer records, click on the navigation buttons on the toolbar.
  • To find a customer record, select Find from Edit menu.
  • To find a next customer record to match your previous condition, select Find Next from Edit menu.
  • To view and modify additional contacts for this customer, select Contacts from combo box on the toolbar.
  • To view and modify notes for this customer, select Notes from combo box on the toolbar.
  • To export customer records to a text file, select Export from File menu.
