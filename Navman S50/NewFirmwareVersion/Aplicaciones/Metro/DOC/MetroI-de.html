Métro 
Installation

Achtung! Datenbanken aus älteren Versionen von "Métro" (< 4.2.0) können nicht mehr mit der aktuellen Version verwendet werden! Es wird empfohlen, daß Sie jede Version von "Métro" ältere als 4.2.0 löschen, bevor Sie diese Version installieren (Eintrag "Löschen" im Systemmenü "Anwendung" auf PalmOS 3.0).

Download

Als erstes müssen Sie das Archiv aus Programm und den Städtedateien downloaden: siehe die Homepage.

Kompatibilität

Palm

Métro  ist kompatibel mit allen PDAs mit PalmOS 2.0 oder neuer (Geräte von Palm/PalmOne, Handspring, Handera, Sony, ...). Die Software ist nicht kompatibel mit dem originalen Pilot 1000 & Pilot 5000.

Métro  ist optimiert für Geräte mit PalmOS 5.x (fast alle momentanen Modelle) und benutzt auf diesen PDA eine vielfach schnellere Engine (alle Berechnungen werden in beinahe Nullzeit erledigt). Es ist ebenso auf die hochauflösenden Bildschirme der momentanen Geräte von PalmOne und Sony optimiert: die 320x320 oder 320x480 Bildpunkte dieser Geräte vollständig ausgenutzt, inklusive 480x320 Landscape-Modus.
Anmerkung: für den 320x480-Modus des PalmOne Tungsten T3 werden zusätzliche Dateien benötigt (siehe die Homepage).

PocketPC

Métro  ist kompatibel mit PDAs (entweder im "Pocket PC"- oder "Handheld PC"-Format) die mit dem Microsoft® Betriebssystem Windows CE (Versionen 2.01, 2.11 und 3.0), Windows Mobile for PocketPC (Versionen 2000, 2002, 2003, 2003 SE und 2005, inklusive "Phone Edition") und Windows Mobile 6 Professional laufen: Geräte von Compaq, HP, Qtek, HTC, Toshiba, Siemens, Casio...

MS Smartphone

Métro  is kompatibel mit Telefonen die mit dem Microsoft® Smartphone Betriebssystem laufen.

Für diese Smartphones lesen Sie bitte die Datei "ReadmeMe", die im zugehörigen ZIP-Archiv enthalten ist für weitere Installationsanweisungen.

Symbian

Métro  is kompatibel mit Telefonen die mit dem Symbian UIQ Betriebssystem (SonyEricsson P800 & P900, Motorola A920 & A925, ...), Symbian Series 60 (Nokia) laufen, Symbian Series 80 (Nokia) laufen oder Nokia Symbian Series 90 Version laufen.

Für diese Smartphones lesen Sie bitte die Datei "ReadmeMe", die im zugehörigen ZIP-Archiv enthalten ist für weitere Installationsanweisungen.

Entpacken

Sie sollten Métro  als komprimierte Datei mit dem Namen "metro.zip" erhalten haben. Sie sollten die Dateien nun aus dem Archiv auf Ihren Desktop-Computer entpacken. Sie können eines der vielen dafür verfügbaren Tools verwenden: WinZip für Windows oder StuffIt Expander für Macintosh (sehr empfohlen, andere Tools können die Dateien beschädigen).

Installation

PalmOS (Palm, Sony, Handera, Handspring, ...)

  • Wählen Sie die Programmdatei in Ihrer Sprache (beispielsweise: metro-de.prc für die deutsche Version). Installieren Sie nur eine Programmdatei!
  • Wählen Sie die Städte, die Sie verwenden wollen, aus dem richtigen Verzeichnis (in den meisten Fällen /all). Sie müssen mindestens eine Stadt installieren!
  • Installieren Sie die ausgewählten Dateien (Programm und Städte) auf Ihrem Gerät (benutzen Sie das Install Tool oder die "Desktop Software", die mit Ihrem PDA geliefert wurde oder klicken Sie einfach doppelt auf jede Datei).
  • Synchronisieren Sie Ihr Gerät, um die Dateien zu installieren.
  • Das Icon ist jetzt auf Ihrem Palm verfügbar: "tippen" Sie es an, um das Programm zu starten. Bei der ersten Verwendung werden Sie aufgefordert, ein Städtenetz auszuwählen.

Installation auf einer Speicherkarte: Wenn Ihr Gerät über eine Speichererweiterungskarte verfügt, können Sie die Métro-Dateien dort installieren indem Sie wie folgt verfahren:

  • Für die Programmdateien (metro-xx.prc): Wählen Sie im Programm "Install Tool" (es ist beim Palm Desktop enthalten) einfach das Ziel "Speicherkarte". Die Installation des Programms auf einer Speicherkarte wird nicht empfohlen.
  • Für die Städtedateien (xxx.pdb): Das Programm findet die Städte nur, wenn sie in einem bestimmten Verzeichnis auf der Karte vorhanden sind. Um die Dateien ins korrekte Verzeichnis zu installiern, haben Sie mehrere Möglichkeiten:
    • Installieren Sie die Dateien im Hauptspeicher (mit dem Install Tool), benutzen Sie dann die "Übertragen" Funktion in Métro  um sie auf die Karte zu verschieben (siehe Anleitung). NICHT MIT "GROßEN" DATEIEN vor allem die Paris-full- und NewYork-works Datenbanken).
    • Installieren Sie mein Dienstprogramm Folder auf Ihrem Palm. Es wird dafür sorgen die Dateien nach der Installation auf der Karte dorthin zu schieben, wo sie hingehören.
    • Installieren Sie die Dateien mit dem Install Tool (wählen Sie "Speicherkarte" als Ziel) auf die Speicherkarte und verschieben Sie sie dann mit Hilfe eines Programms wie FileZ manuell ins Verzeichnis /PALM/Programs/metro.
    • Verwenden Sie einen Kartenleser (oder eine Software wie CardExport) um die Dateien an Ihrem PC ins Verzeichnis /PALM/Programs/metro auf derKarte zu kopieren.

Windows Mobile for PocketPC & Smartphone (Compaq, HP, Qtek, HTC, Toshiba, Casio, ...)

  • Auf Ihrem Desktop PC: Starten Sie das setup.exe-Programm im Windows-Verzeichnis, um das Programm auf Ihrem Gerät zu installieren
  • Wählen Sie die Städte, die Sie verwenden wollen aus dem richtigen Verzeichnis (in den meisten Fällen /all). Sie müssen mindestens eine Stadt installieren!
  • Klicken Sie auf das Icon "Mein Computer"
  • Wählen Sie unter den verfügbaren Geräten dasjenige, welches mit "Mobiles Gerät" benannt ist
  • Dort können Sie die Verzeichnisse und Dateien Ihres Gerätes sehen
  • Kopieren Sie das oben ausgewählte Programm und die Städte in ein Verzeichnis Ihrer Wahl
  • Auf Ihrem Gerät: klicken Sie auf Métro , wählen Sie Ihre Sprache unter "Voreinstellungen" im "Optionsmenü"
  • Wählen Sie "Network" im "Optionsmenü", um die zu verwendende Stadt auszuwählen

Sie können das Programm auch manuell installieren, anstatt das Setup-Programm zu benutzen (beziehungsweise werden es sogar müssen, wenn Sie ein Gerät mit WinCE 2.0 oder 2.01 einsetzen, siehe Ende der folgenden Tabelle):

  • Wählen Sie die richtige Cabinet-Datei (.cab) für Ihr PocketPC-Gerät aus der nachfolgenden Tabelle
  • Klicken Sie auf Ihrem Desktop PC auf das Icon "Mein Computer"
  • Wählen Sie unter den verfügbaren Geräten dasjenige, welches mit "Mobiles Gerät" benannt ist
  • Dort können Sie die Verzeichnisse und Dateien Ihres Gerätes sehen
  • Für ein .cab Datei:
    • Kopiern Sie die ausgewählte .cab Datei in ein Verzeichnis Ihrer Wahl
    • Auf Ihrem PDA öffnen Sie den Explorer, gehen sie in das Verzeichnis, in das Sie die Datei kopiert haben
    • Klicken Sie auf die Datei, das Programm wird installiert und ein Eintrag generiert
  • Für ein .exe Datei: kopiern Sie die ausgewählte .exe Datei auf Ihrem PDA

Symbian (Nokia, SonyEricsson, ...)

  • Stellen Sie sicher, das richtige Archiv für Ihr Telefon herunterzuladen (Hilfe hier).
  • Kopieren Sie die .sis-Datei aus dem Archiv auf Ihr Telefon (Übertragung mit Hilfe des Programms des Herstellers oder über Infrarot oder Bluetooth)
  • Starten Sie die .sis-Installation auf Ihrem Telefon (siehe Handbuch für nähere Informationen)

Wenn Sie während der Installation einen Sicherheitshinweis erhalten (der aussagt, die Software ist nicht "signiert"), rufen Sie im "Werkzeuge"-Programm "Parameter" unter "Optionen" auf. Schalten Sie im Fenster "Softwareinstallation" die Einstellung "Nur signierte" aus. Dann sollte die Installation des Programms möglich sein.
Die genaue Lage der Einstellungen kann je nach Konfiguration Ihres Gerätes abweichen: sehen Sie deshalb bitte in Ihrem Gerätehandbuch für nähere Informationen nach.

Die Symbian-Archive enthalten nur einen Teil der Datenbanken (nur die großen Städte). Wenn Sie weitere Städte installieren wollen, laden Sie bitte das Komplettpaket (metro.zip) herunter und kopieren Sie die von Ihnen gewünschte .pdb-Datei auf Ihr Telefon.
