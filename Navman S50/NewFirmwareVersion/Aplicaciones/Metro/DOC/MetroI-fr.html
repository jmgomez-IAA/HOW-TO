Compatibilité

Palm

Métro  est compatible avec tous les assistants personnels équipés de PalmOS 2.0 ou supérieur (appareils des constructeurs Palm/PalmOne, Handspring, Handera, Sony, ...). Le logiciel n'est donc pas compatible avec les Pilot 1000 et Pilot 5000 d'origine.

Métro  est optimisé pour les appareils équipés de PalmOS 5.x (la quasi-totalité des appareils récents) et bénéficie sur ces machines d'une vitesse de calcul accrue (les recherches sont presque instantanées). Il est également optimisé pour les écrans en haute résolution des appareils récents de PalmOne et Sony (toutes générations) : il profite pleinement des résolutions 320x320 et 320x480 de ces Palms, y compris le mode paysage 480x320.
Note : l'utilisation de la totalité de l'écran sur Tungsten T3 requiert l'installation de fichiers additionnels (voir en page d'accueil).

PocketPC

Métro  est compatible avec les assistants personnels au format "Pocket PC" ou "Handheld PC" équipés des systèmes d'exploitation de Microsoft® Windows CE (versions 2.01, 2.11 et 3.0), Windows Mobile for PocketPC (versions 2000, 2002, 2003, 2003 SE & 2005, y compris "Phone Edition" i.e. smartphone AVEC écran tactile) & Windows Mobile 6 Professional : appareils des constructeurs Compaq, HP, Qtek, HTC, Toshiba, Siemens, Casio...

MS Smartphone

Métro  est compatible avec tous les téléphones équipés du système Microsoft® Smartphone.

Pour ces smartphones, consultez le fichier "readme" inclus dans le fichier zip correspondant.

Symbian

Métro  est compatible avec tous les téléphones équipés du système Symbian UIQ (SonyEricsson P800 et P900, Motorola A920 et A925, ...), Symbian Series 60 (Nokia et autres), Symbian Series 80 (Nokia et autres) ou Symbian Series 90 de Nokia.

Pour ces smartphones, consultez le fichier "readme" inclus dans le fichier zip correspondant.

Extraction

Vous devez maintenant extraire le contenu des fichiers sur votre ordinateur de bureau. Utilisez pour ce faire un des nombreux outils disponibles : par exemple WinZip sous Windows ou StuffIt Expander sur Macintosh (expressément recommandé sur Macintosh, les autres outils pouvant provoquer des anomalies).

Installation

Assistants Palm (Palm, Sony, Handera, Handspring, ...)

  • Choisissez dans la liste ci-dessus le programme correspondant à votre langue (exemple : metro-fr.prc pour la version française). N'installez qu'une seule version du programme !
  • Choisissez les villes que vous souhaitez utiliser dans le répertoire qui vous convient (répertoire all dans la majorité des cas). Il faut impérativement installer au moins une ville !
  • Demandez l'installation de ces fichiers sur votre assistant (utilisez l'outil d'installation livré avec votre assistant, ou l'application "Palm Desktop" ou simplement en double-cliquant sur chacun des fichiers).
  • Effectuez une synchronisation (HotSync) de votre assistant pour y installer les fichiers.
  • L'icône est maintenant présente sur le sélecteur d'applications de votre Palm : "cliquez" dessus pour lancer l'application. Lors de la première utilisation vous devrez sélectionner la ville à utiliser.

Installation sur une carte mémoire : si vous disposez d'une carte d'extension mémoire, vous pouvez y installer les fichiers de Métro , en respectant ces consignes :

  • Pour le programme (fichier metro-xx.prc) : choisissez simplement la destination "Carte d'extension" dans l'outil "Installation rapide" de Palm Desktop. L'installation du programme sur carte mémoire n'est cependant pas recommandée.
  • Pour les villes (fichier xxx.pdb) : le programme ne trouvera les villes que si elles sont présentes dans un répertoire spécifique de votre carte mémoire. Pour installer les fichiers à l'emplacement adéquat, vous disposez de plusieurs options :
    • Installez les fichiers en mémoire principale (avec l'outil d'installation rapide), puis utilisez l'option de "transfert" de Métro  pour les déplacer sur la carte mémoire (cf. le mode d'emploi). NE TENTEZ PAS CETTE OPTION AVEC LES "GROS" FICHIERS (tels que Paris-full ou NewYork-works).
    • Installez mon utilitaire Folder sur votre Palm. Il se chargera de déplacer les fichiers là où il faut après une installation rapide sur carte mémoire.
    • Installez les fichiers sur la carte mémoire avec l'outil d'installation rapide (en choisissant la destination "Carte d'extension") puis déplacez manuellement les fichiers dans le répertoire /PALM/Programs/metro avec un outil tel que FileZ.
    • Utilisez un lecteur de carte mémoire (ou un logiciel comme CardExport) et, depuis votre PC, copiez les fichiers dans le répertoire /PALM/Programs/metro de la carte.

Assistants Windows Mobile PocketPC & Smartphone (Compaq, HP, Qtek, HTC, Siemens, Toshiba, Casio, ...)

  • Sur votre ordinateur de bureau, exécutez le programme setup.exe du répertoire Windows pour installer le programme sur votre appareil.
  • Choisissez les villes que vous souhaitez utiliser dans le répertoire qui vous convient (répertoire all dans la majorité des cas)
  • Ouvrez l'icône "Mon Ordinateur" ("My Computer")
  • Parmi la liste des lecteurs disponibles, ouvrez l'icône "Dispositif Mobile" ("Mobile Device")
  • Vous pouvez voir les dossiers existants sur votre assistant
  • Copiez les villes que vous avez sélectionnées dans le répertoire de votre choix
  • Sur l'appareil, cliquez sur le raccourci de Métro , choissez votre langue dans l'option "Préférences" du menu "Options"
  • Choisissez votre ville par l'option "Réseaux" du menu "Options"

Vous pouvez également installer le programme manuellement, sans utiliser le programme "setup" (et cela sera nécessaire si vous avez un appareil équipé de WinCE 2.0 ou 2.01, consultez la note en bas du tableau) :

  • Sélectionnez le "cabinet" (fichier .cab) ou l'exécutable (fichier .exe) correspondant à votre assistant dans la liste ci-dessous :
  • Sur votre ordinateur de bureau, ouvrez l'icône "Mon Ordinateur" ("My Computer")
  • Parmi la liste des lecteurs disponibles, ouvrez l'icône "Dispositif Mobile" ("Mobile Device")
  • Vous pouvez voir les dossiers existants sur votre assistant
  • Si le fichier correspondant à votre appareil est un .cab :
    • Copiez le fichier .cab que vous avez sélectionné dans le répertoire de votre choix
    • Sur votre assistant, ouvrez l'explorateur ("Explorer") et naviguez vers le répertoire où vous avez installé le fichier cab
    • Cliquez sur le fichier, le programme est installé et un raccourci est créé
  • Si le fichier correspondant à votre appareil est un .exe, copiez celui-ci dans le répertoire des programmes :: il vous permet d'exécuter directement Métro (vous pourrez également créer un raccourci à votre convenance)

Symbian (Nokia, SonyEricsson, ...)

  • Assurez-vous d'avoir téléchargé le package correspondant à votre téléphone. Si vous ne connaissez pas le système utilisé par votre smartphone (s60, s80, s90 ou UIQ), consultez cette page.
  • Copiez sur votre téléphone le fichier .sis contenu dans le package (avec les utilitaires livrés avec votre téléphone ou par infrarouge ou bluetooth)
  • Lancez l'installation du fichier .sis (ou .sisx) sur votre téléphone (consultez le manuel de votre téléphone pour plus d'informations).

Si vous recevez une alerte de sécurité lors de l'installation (parce que le logiciel n'est pas "signé"), entrez dans les "Outils" et sélectionnez "Paramètres" parmi les "Options". Dans la fenêtre "Installation de logiciel", désactivez l'option "Signé uniquement". Vous pourrez ensuite lancer l'installation.
La procédure exacte peut dépendre de votre configuration : consultez le manuel de votre téléphone pour plus de détails.

Les packages Symbian contiennent uniquement un échantillon des bases de données (pour les principales capitales). Si vous voulez installer d'autres villes, télécharger le package complet (metro.zip) et copiez les fichier .pdb de votre choix sur votre téléphone.

Mise à jour

Sauf indication contraire (cf. "Attention" ci-dessous), aucune précaution particulière n'est nécessaire pour mettre à jour votre version du programme ou les fichiers de villes : installez simplement les nouveaux fichiers à la place des anciens et c'est tout.

Attention! Les fichiers de villes des versions antérieures à la 4.2.0 ne peuvent être utilisés avec les nouvelles versions. Si vous avez une version plus ancienne, vous devez supprimer tous ses fichiers avant d'installer la nouvelle.
