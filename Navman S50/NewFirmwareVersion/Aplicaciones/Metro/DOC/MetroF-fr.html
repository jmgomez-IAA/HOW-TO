Métro 
Foire aux questions

Installation - Mon ordinateur de bureau est un Macintosh et le programme ne fonctionne pas sur mon Palm : les programmes de décompression des fichiers .zip ont tendance à corrompre les fichiers. Si vous constatez un problème, utilisez Stuffit Expander pour ouvrir le fichier metro.zip.

Général - Pourquoi n'y a-t-il pas les plans des lignes dans le logiciel ? Parce que cela représente beaucoup trop de travail et que nous ne pensons pas que cela soit indispensable. Merci de ne pas insister sur ce sujet !

Général - La liste des stations d'une ligne comporte deux fois le même arrêt : cette ligne comporte plusieurs branches à partir de cette station.

Calcul - Le programme indique "station fermée à l'horaire sélectionné" alors que je sais qu'elle est ouverte : vous avez peut-être désactivé les lignes qui passent par cette station ?

Calcul - J'ai demandé un trajet entre deux stations et le programme indique que le trajet ne peut être calculé : vous avez peut-être désactivé une ligne dans la liste ? Sinon, cela peut être du à une station de départ ou d'arrivée se trouvant sur un tronçon à sens unique qui rend le calcul trop complexe. Essayez la station parcourue en sens inverse, vous obtiendrez certainement un résultat.

Calcul - Le trajet le plus court est plus long que le trajet avec le minimum de correspondances : le programme prend en compte un temps d'attente supplémentaire dans le cas de trains peu fréquents mais il n'est pas inclus dans l'estimation du temps de trajet ; si vous choisissez le trajet avec le minimum de correspondances, vous risquez d'attendre plus longtemps.

Calcul - Le calcul du trajet est très long (par exemple à Genève ou Philadelphie) : beaucoup de paramètres affectent la vitesse de calcul et la puissance limitée des PDA ne permet pas les miracles ; vous pouvez toujours arrêter un calcul en "cliquant" sur la barre de progression...

Résultats - Les directions à New-York n'utilisent pas le vocabulaire local ("uptown", downtown"...) : cette évolution est en cours (déjà prête pour Londres).

Résultats - Les couleurs des lignes ne sont pas fidèles : la gestion des couleurs sur nos appareils n'est pas parfaite, il n'y a malheureusement pas de solution (et ce n'est pas une erreur de codage ;-).

Palm - Installation - Le programme fonctionne dans une langue "étrange" : Vous n'avez pas installé la version correcte du programme. Veuillez lire attentivement le guide d'installation.

Palm - Le programme crashe au démarrage / lors d'un calcul... : Si la première question (cf. Macintosh) n'est pas applicable, vous utilisez probablement une ancienne version d'une ville avec un programme récent (ou l'inverse). Das certains cas, les anciens fichiers ne sont pas remplacés par les nouveaux. Vérifiez la liste des villes ou, plus facile, supprimez totalement le programme et les villes et ré-installez.

Palm - Le programme crashe au démarrage / lors d'un calcul... : Si la solution précédente ne vous convient pas, vérifiez que votre appareil dispose d'un peu de mémoire libre pour le fonctionnement du programme. Dans le cas contraire, supprimez quelques villes ou les programmes dont vous ne vous servez pas.

Palm - Le contraste est très faible sur mon Palm IIIe ou IIIex. Le programme fonctionne en niveaux de gris et ces machines sont connues pour mal supporter ce mode. Revenez au mode noir et blanc en cochant l'option dans les préférences (et redémarrez le programme pour voir la différence).

Palm - Puis-je utiliser les réseaux installés sur une carte d'extension ? Oui, cela est possible mais les fichiers doivent être copiés impérativement dans un répertoire précis sur la carte (voir le guide d'installation pour plus de détails).

Palm - Lors de la synchronisation de certaines villes, j'obtiens une erreur et le fichier n'est pas installé sur mon Palm : Les "gros" fichiers (actuellement, Paris-full et NewYork-works) ne peuvent être installés que sur une carte mémoire externe.

Palm - Le programme affiche "Aucune ville installée" au démarrage : Voir le point précédent, ou lisez attentivement le guide d'installation.

Windows Mobile - Smartphone - Le programme a un comportement anormal : Microsoft propose deux systèmes distincts et incompatibles pour les smartphones, à savoir "Windows Mobile for PocketPC Phone Edition", pour lequel vous devez utiliser notre version "PocketPC" de Métro et "Windows Mobile for Smartphone" pour lequel il vous faut notre version "MS Smartphone" de Métro. Vérifiez que vous avez installé la version correspondant à votre système.

Windows Mobile - Une partie de la fenêtre principale "déborde" de l'écran. Installez l'utilitaire "Regking" (disponible gratuitement sur PocketGear) ; exécutez ce programme ; choisissez votre format de police ; réinitialisez votre assistant. Information fournie par V. GAKH (merci !).

Windows Mobile - Comment lancer Métro  en double-cliquant sur le fichier d'une ville ? Utilisez "Palm Tweak" (ou un utilitaire équivalent) pour associer l'extension de fichier "pdb" au programme "Métro".

Windows Mobile - Comment changer de police de caractère dans le programme ? Copiez les fichiers de police dans le répertoire windows/fonts et sélectionnez votre police dans "Métro".
