Frequently Asked Questions - Häufig gestellte Fragen

Installation - Mein Desktop Computer ist ein Macintosh und das Programm stürzt ab: viele Entpacker neigen dazu, den Inhalt von .zip Dateien zu zerstören. Wenn Sie ein Problem haben, benutzen Sie Stuffit Expander (kostenlos), um metro.zip zu entpacken.

Allgemein - Warum enthält das Programm keine Karten? Es ist zu viel Arbeit für uns und wir denken auch nicht, daß das unbeding nötig ist. Bitte bestehen Sie nicht darauf!

Allgemein - Die Liste der Haltestellen zeigt manche Einträge mehrfach: Die Linie hat mehrere Zweige ab dieser Haltestelle.

Berechnung - Das Programm zeigt an, daß "die Station zu der gewählten Tageszeit geschlossen ist" aber ich bin sicher, sie ist geöffnet: Haben Sie die Linie, die diese Haltestelle anfährt, vielleicht abgewählt?

Berechnung - Ich habe eine Route abgefragt, das Programm sagt jedoch, sie könne nicht berechnet werden: Haben Sie vielleicht eine Linie "abgewählt" die verhindert, daß die Route berechnet werden kann? Oder Sie haben vielleicht eine Haltestelle ausgewählt, die nur in einer Richtung angefahren wird, was dazu führen kann daß die Berechnung zu komplex wird. Versuchen Sie, die Haltestelle in der anderen Richtung zu finden und sie werden ein Ergebnis erhalten.

Berechnung - Die geschätzte Zeit für die kürzeste Route ist aber länger als die für die direkte Verbindung: Das Programm kann eine extra Wartezeit für besonders unregelmäßig verkehrende Linien einrechnen, die nicht in der geschätzten Zeit enthalten ist. Wenn Sie die Alternative der "direkten Verbindung" nutzen, kann es zu Wartezeiten an einem Halt kommen.

Berechnung - Das Programm braucht lange, um eine Route zu berechnen: Viele Faktoren können die Geschwindigkeit des Programms beeinflussen und die begrenzte Geschwindigkeit der PDA erlaubt leider keine Wunder. Sie können die Berechnung jederzeit anhalten, indem Sie auf den Fortschrittsbalken "tippen" ...

Ergebnisse - Die Richtungen in New York benutzen nicht die dort üblichen Begriffe ("uptown", downtown" ...): An diesem Feature wird noch gebastelt (schon verfügbar in London).

Ergebnisse - Die Linienfarben scheinen entstellt zu sein: Diese kleinen Geräte haben nur begrenzte Farbmanagement-Fähigkeiten und daran kann man nichts ändern (und das ist kein Fehler unsererseits ;-) ).

Palm - Installation - Das Programm läuft in einer "seltsamen" Sprache: Sie haben die falsche Sprachversion des Programms installiert. Bitte lesen Sie die Installationshinweise aufmerksam.

Palm - Das Programm stürzt beim Start / wenn ich eine Berechnung ausführe ab ...: Wenn der erste Tip (über den Macintosh) nicht hilft, ist es wahrscheinlich, daß Sie eine alte Datenbank mit einem neuen Programm (oder umgekehrt) benutzen. In einigen Fällen werden alte Dateien nicht ersetzt, wenn Sie neue installieren und das Programm benutzt weiterhin die falschen. Überprüfen Sie die Städteliste oder, einfacher, löschen Sie Programm und Städte komplett und installieren Sie neu.

Palm - Das Programm stürzt beim Start / wenn ich eine Berechnung ausführe ab ...: Wenn der vorhergehende Tip nichts nützt, überprüfen Sie, ob Sie genug freien Speicher in Ihrem Computer haben um das Programm laufen zu lassen. Wenn nicht, versuchen Sie ein paar Städte zu löschen, oder andere Programme, die Sie nicht mehr benötigen.

Palm - Der Kontrast auf meinem Palm IIIe/IIIex ist sehr niedrig. Das Programm nutzt einen Graustufen-Modus und diese Geräte sind dafür bekannt, damit Schwierigkeiten zu haben. Schalten Sie in den Schwarz/Weiß-Modus in den Einstellungen (und starten Sie das Programm neu, um den Unterschied zu sehen).

Palm - Kann ich die Städtedatenbanken auf einer Erweiterungskarte installieren? Ja, das ist möglich, aber Sie müssen die Dateien genau dorthin kopieren, wo sie, wie in der Installationsanleitung beschrieben, hingehören.

Palm - Beim Aufspielen mancher Städte bekomme ich einen Fehler und die Datei wird nicht auf meinem Palm installiert: Unsere größten Dateien (mementan Paris-full und NewYork-works) können nur auf Speicherkarte installiert werden.

Palm - Eine Meldung sagt beim Start "Keine Stadt installiert": Siehe den vorherigen Eintrag oder lesen Sie die Installationshinweise aufmerksam.

Windows Mobile - Smartphone - Das Programm zeigt ein seltsames Verhalten: Microsoft bietet zwei verschiedene und inkompatible Systeme für Smartphones. Eine ist "Windows Mobile for PocketPC Phone Edition", welches die "PocketPC"-Version von MetrO benötigt, das andere ist "Windows Mobile for Smartphone", für welches Sie die "MS Smartphone"-Version von MetrO benötigen. Prüfen Sie ob sie die richtige Version für Ihr System installiert haben.

Windows Mobile - Ein Teil des Hauptfensters überschreitet die Bildschirmränder: Installieren Sie das Utility "Regking" (frei erhältlich beispielsweise bei PocketGear); und starten Sie es; wählen Sie eine Schriftart; resetten Sie Ihr Gerät. Information provided by V. GAKH (danke!).

Windows Mobile- Wie kann ich Métro  dazu bringen, bei Doppelklick auf eine Stadtdatei zu starten? Benutzen Sie "Palm Tweak" (oder ein ähnliches Utility), um die Dateiendung "pdb" mit dem "Métro" Programm zu verbinden.

Windows Mobile - Wie kann ich die Schriftart, die vom Programm verwendet wird, ändern? Kopieren die gewünschte Schriftartendatei ins Verzeichnis windows/fonts und wählen Sie sie in "Métro" aus.
