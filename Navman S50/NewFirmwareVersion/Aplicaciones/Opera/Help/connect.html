Connecting to the Web

If your Pocket PC does not already have a working Internet connection, you will need to add one to run Opera. This is the connection Opera will use when you launch the browser.

Create a GPRS Connection

  1. From the home screen, select Start > Settings > Connections > Connections.
  2. Select "Add a new modem connection".
  3. Enter a name for the connection and select "Cellular line (GPRS)" as the modem type.
  4. Tap "Next" and under "Access point name", enter the access point information provided by your mobile operator.
  5. Tap "Next" and enter any other information provided by your mobile operator (tap "Advanced" for more options).
  6. Tap "Finish" to create the connection.
  7. Go back to "Connections" and use "Manage existing connections" to make sure the connection you just configured is set as default.

Note: If you are not sure about which information to enter, ask your mobile operator or refer to www.opera.com/connect. If your connection settings do not have the "Cellular line (GPRS)" or an equivalent option available, then you may be able to use other options to connect via a mobile phone, such as with Bluetooth or infrared, or you may be able to use your USB PC connection to connect to the Internet.

Tip: Visit Opera's on-line guides with your desktop browser.
