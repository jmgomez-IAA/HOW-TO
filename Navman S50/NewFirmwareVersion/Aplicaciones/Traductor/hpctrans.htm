HPC Translate is an utility that allows the user to translate single words from one language to another. It runs on Microsoft Windows-Powered Pocket PCs, Handheld PCs and Palm-Size PCs. HPC Translate includes English and Spanish dictionaries. These dictionaries contain approximately 32,000 words each. HPC Translate can translate in both directions, i.e. you may enter a word in the From field and tap the >> button. HPC Translate can also translate a selected word in almost any Windows CE application, such as Pocket Word, Ink Writer, Palm-Size PC Notetaker, HPC Notes, etc. Note: this feature is not supported on Pocket PCs. See Using HPC Translate section for more information.

If you are interested in support for other languages, you can purchase additional dictionaries for HPC Translate. German and French dictionaries are available now. For more information and to purchase additional dictionaries, please visit our web site at http://www.phatware.com/hpctrans.html.

Using HPC Translate

To translate a selected word:
(not supported on Pocket PCs)

You may use HPC Translate to translate a selected word in almost any Windows CE application that uses standard edit control, such as Pocket Word, Ink Writer, Palm-Size PC Notetaker, HPC Notes, Email Editor, and may others. To do so:

  • Make sure that HPC Translate is running, i.e. HPC Translate icon appears in the task bar.
  • Select the desired word in the editor that you are currently using.
  • Tap the HPC Translate icon in the Windows CE Task Bar.
  • HPC Translate main window will appear and the selected word will be copied into the From field.
  • If the word is found in the dictionary the translation of the word will appear in the To edit box. If the word has more than one translation, all possible translations will appear in the To list box.

After the word is successfully translated, you may replace the original word in the document by the translated one simply by taping on Replace button in the toolbar, or choosing Edit/Replace menu item, or using Ctrl+R keyboard shortcut. Please note, that Replace command may not always be available.

To translate a word:

  • Select the desired language from which you want to translate in the From drop-down menu
  • Select the desired language to which you want to translate in the To drop-down menu
  • Enter or paste the desired word into the From edit box
  • Tap the >> button, or use Action/Translate >> menu command, or use Ctrl+T keyboard shortcut.

If the word is found in the dictionary the translation of the word will appear in the To edit box. If the word has more than one translation, all possible translations will appear in the To list box. You may copy the result of your translation to the clipboard using Edit/Copy or Edit/Copy All menu commands.

If the word is not found in the dictionary a message will appear in the Status bar. If the word doesn't exist in the dictionary, but you know the translation, you may add it to the user-defined dictionary.

To add a new translation to the user-defined dictionary:

  • Enter the word in the selected language into From edit box
  • Enter the translation of the word into the To edit box
  • Tap Add button, or use Edit/Add Word menu item, or use Ctrl+W keyboard shortcut.

A new pair of words will be added to the user dictionary ENG_ESP.UDR. This is a plain text file that has the following format:

English:Spanish
word11:word12
word21:word22
.............

You may edit this file using any text editor, such as Pocket Word, to add or remove user-defined translations.

Setting HPC Translate Options

To display Options dialog box use Tools/Options or Options button in the toolbar or Ctrl+O keyboard shortcut.

To modify Translator Options:

  • Case Insensitive - check this box if you want to ignore case for words with mixed capitalization.
  • Look in User Dictionary - check this box if you want to search for possible translation in the user dictionary if the word was not found in the main dictionary.
  • Smart Search - check this box to use Smart Search. This option may be disabled if not supported by the selected dictionary. Smart search uses grammatical rules to find a similar word, if the word doesn't exist in the dictionary. For example, it you enter the word tested and this word is not in the dictionary, the word test will be translated instead.
  • Use Spell Checker - if word is not found in the dictionary or misspelled, HPC Translate will try to find a similar word in the dictionary. This option may be disabled if not supported by the selected dictionary.
  • Disable Automatic Character Conversion - if you have a mobile device with European keyboard you may disable automatic character conversion feature. See Entering non-ASCII Characters section for more information. Note: recommended on Palm-Size PC if you use Jot to input characters.

To change the Dictionaries Path:

All dictionaries must be located in the same folder. For example, Spanish/English dictionaries require five files:

  • ENG.TDD - Main English dictionary
  • ESP.TDD - Main Spanish dictionary
  • ENG_ESP.TDL and ESP_ENG.TDL - Translation tables for English/Spanish translator.
  • ENG_ESP.UDR - User-defined dictionary for English/Spanish translator. This file is created automatically by HPC Translate when required.

If you move these files from the default folder, enter a new folder into Dictionaries Path edit box. Tap <...> button to use File Open dialog box to browse for a specific folder.
