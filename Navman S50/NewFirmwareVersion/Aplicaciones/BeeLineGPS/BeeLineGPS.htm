GPS Tab

The GPS page shows GPS status. It shows signal quality, a choice of current position, DOP values and the azimuth/elevation for individual satellites in view. Tapping on the page will change screens from current position to DOP values to azimuth/elevation.

Panel Page (tab)

The Panel Page shows various types of telemetry represented in an analog configuration. Use the tap and hold menu to setup this page.

Tap and Hold Menu
  • Cycle through gauges
    Select this menu item to cycle through each of the gauges that are selected. When in cycle mode, there will be only one gauge displayed as big as the screen will allow. If this is not selected, the screen will display all of the gauges that are selected in one screen.
  • Altimeter
    Select this menu item to enable or disable the Altimeter.
  • Compass
    Select this menu item to enable or disable the Compass.
  • Speed
    Select this menu item to enable or disable the Speedometer.
  • Vertical Speed
    Select this menu item to enable or disable the Vertical Speed indicator.
  • Setup Gauges
    • Speed...
      Sets up the maximum or minimum speed shown on the Speedometer.
    • Enable Smooth Mode
      Enable/disables gauge smooth mode. This mode animates the gauges at about 10 frame per second.
  • Setup Color
    Allows the user to choose form a list of gauge colors
  • Show Nav Window
    Enables/disables navigation window

Tam and Hold Menu
(Nav Window)

  • Change This field...
    Allows the user to choose from a list of trip computer items
  • Show Nav Window
    Enables/disables navigation window
  • Reset Trip Data
    Resets the trip computer

 

Map Page (tab)

The Map Page show waypoints, tracks and the current position as reported by the GPS receiver. Make use of the tap and hold to bring up the popup menu. If the popup menu was invoked on top of a waypoint, then operations may be invoked for the selected waypoint.

When waypoint averaging is enabled an average status window will be shown below the map window. This window shows the status of the current average in progress.

Keys

Use the Joystick key to zoom in and out. By pulling back on the joystick will zoom out and pushing forward on the joystick will zoom in.

Tap and Hold Menu (Map)

  • Goto This Waypoint...
    Enables a Goto from from the selected waypoint when the popup menu was invoked.
  • Edit This Waypoint...
    Edits the waypoint selected when the popup menu was invoked.
  • Delete This Waypoint...
    Deletes the selected waypoint
  • Geocache
    • Found it
      Forces the geocache state to "Found" and updated the found time to the current time.
    • Not Found
      Forces the geocache state to "Not Found"
    • Show Geocache Info...
      Shows the details in HTML of the geocache as returned by the Geocaching.com GPX file.
  • Mark Waypoint Here
    Marks a new waypoint where the popup menu was invoked.
  • Center Map Here
    Centers the map to where the popup menu was invoked.
  • Center Map to Current Position
    Centers map to the current GPS position.
  • Set Current Position Here
    Sets the current position to where the tap-and-hold menu was invoked. This is similar to a position update from the GPS receiver but forced manually by the user.
  • Erase Current Track
    Erases and clears the current track.
  • Map Settings
    • North Up
      Forces the map into North Up mode.
    • Course Up
      Forces the map into Course Up mode
    • Map Properties...
      Shows the Map Properties Dialog.
Tap and Hold Menu (Navigation Window)
  • Change Fields...
    Shows the Change Fields Dialog.
  • Hide Window
    Hides the Navigation Window
  • Map Properties...
    Shows the Map Properties Dialog.
Tap and Hold Menu (Altitude Profile Window)
  • Clear Altitude Graph
    Clears and resets the Altitude Graph
  • Hide Window
    Hides Altitude Graph Window.
  • Map Properties...
    Shows the Map Properties Dialog.
Tap and Hold Menu (Waypoint Average Status Window)
  • Stop Average

  • Cancel Average

  • Map Properties

Trip Page

The Trip Page will allow you to setup trip and navigational field for display. Each pane or trip field may be setup by using the tap and hold menu feature and then selecting Change This Field. The page layout may also be changed by the using the tap and hold feature and selecting the Screen Configuration menu item.

Tap and Hold Menu
  • Change This Field...
    Changes the field where the tap and hold was evoked.
  • Screen Configuration
    There are several screen configurations that may be selected. This will allow you to control the number of fields to display. Up to twelve (12) fields may be displayed at one time.
  • Reset Trip Data
    Select this menu item to reset the trip data. This affects items such as the Odometer.

Main Menu (Tools)

  • Configuration
    • Map Properties...
      Select this menu item to show the Map Properties Dialog.
    • Units...
      Select this menu item to show the Units Dialog.
  • GPS
    • Setup Serial Port (Comm Port)
      Select this menu item to select a serial port that the GPS is connect to. See the Serial Port Dialog for more information. A checkmark next to this menu item indicates that BeeLineGPS is using the serial port to connect to the GPS receiver.
    • Playback Logged NMEA File
      Select this menu item to playback a previously recorded NMEA file. A checkmark next to this menu item indicates that BeeLineGPS is using a file for its NMEA data and there is no connection to the GPS receiver.
    • Record NMEA Log
      Select this menu item to start or stop a NMEA log file.
    • Manufacture Specific (more help)
      • Setup SiRF NMEA Output
        Select this menu item if the GPS receiver is using a SiRF GPS chipset. BeeLineGPS will setup the NMEA output.
    • Autodetect GPS...
      Select this menu item if to automatically detect and setup the GPS receiver.
  • Track
    • Export Track...
      Exports the current track to a file. Currently the file export format is comma separated values (CSV).
      The format is: latitude, longitude, altitude, start
      West and South are indicated as negative values. The start parameter indicates that this is a start point of a track and if there is a previous point before the start point, the track line should not connect to it.
  • View
    • Status Bar
      Select this menu item to view or hide the status bar. This is a toggle function where the check box next to the menu item indicates that the item is enabled.
  • Enter License Key...
    Select this item to show the License Key Dialog.
  • About
    Shows software version
  • Help
    This is a local help link. In addition to this link, you can use the system start menu followed by selecting help to access help.
  • Exit Application
    Closes and exits application.

Map Properties Dialog

Use this dialog to change the map orientation and enable/disable specific map details.

Map Tab

  • North Up
    North Up will orient the map such that North is always up.
  • Course Up
    Course Up will orient the map such that the direction of travel is at the top of the map.
  • Waypoint Arrival Alarm Distance
    Use this parameter to adjust when the waypoint arrival is triggered. This applies only to a GoTo waypoint.
  • View Navigation Window
    Enables the Navigation Window on the map. The Navigation Window can show navigation status such as speed, direction and distance to a GoTo waypoint.
  • View Altitude Profile Window
    The Altitude Profile Window will show the altitude graphically as time progresses.
  • Enable Position Filter
    Under normal operation, this item should be checked. When in a stationary position, the GPS position will wonder and the position is changing very slightly. When the filter is enabled, it will take approximately a 5 meter position change before a new track point is recorded and certain trip computer items are updated. When the filter is disabled, every new position is recorded in the track log as well as used in the trip computer.
Averaging Tab
  • Qualify Waypoint Averaging
    Check this to qualify the position against the DOP threshold before using it in the average.
  • HDOP Threshold
    Sets the HDOP threshold. BeeLineGPS will use the position if it is less than or equal to this DOP value. To use positions calculated with good geometry, use a numbers between 1.0 and 5.0. To use positions with relaxed geometry, use numbers in the 10s and 20s.

Units Dialog

Use the Units Dialog to setup measurement units for the application.

  • Distance/Speed
    Set to the desired units for your application.
  • Altitude/Elevation
    Set to the desired units for your application.
  • Position Format
    Set to the desired units for your application.

Serial Port Dialog

Please note that a Bluetooth connection typically has two ports associated with it. There is a transmit port and a receive port. The port number and description will vary from one manufacturer to another. Typically connect to the receive port.

License Key Dialog

The License Key Dialog is where the license key is entered. Enter the key exactly as it appears. The key is case sensitive.

Edit Waypoint Dialog

The Edit Waypoint Dialog is comprised using several tabbed windows. These categories are described as follows.

Settings

The Settings tab hold the position information, altitude and the icon associated with the waypoint. The position format will follow the same format as selected in the Units Dialog. 

Active Waypoint

Active Waypoints are waypoints that will trigger a sound file when within a trigger distance.  Check the "Active" check box to enable the waypoint to be active. Set the trigger distance to the desired distance to trigger the sound file. Currently only wav file are supported.

Geocache

The Geocache tab helps manage the waypoint as a geocache. There are several states associated with a geocache. If the "Set icon to Geocache State" is selected, then the waypoint icon will follow the geocache state. To change the geocache state, select the state from the "Geocache State" dropdown box. 

NOTE: If  "Geocache not found" is selected and "Set icon to Geocache State" is selected as well, the displayed icon may take on several automatic states. These states are: "New and not found" and "Future cache" depending on the place date on the cache.

The Geocache Found on parameter is designed for you to set the date that the cache was found. If this date has never been set and you are visiting this tab for the first time for the particular cache, then the current data is used default.

Quick Start Guide

All screens have an alternate menu associated with them. Tap and hold on the particular screen to view the alternate menu.

Map

  • Tapping on a waypoint will show more information about that waypoint as well as the direction and distance from the current position.
  • Tap and hold on a waypoint will allow you to delete, edit, Center the map to and go to the selected waypoint.
  • Tap and drag will pan the map. While dragging, there is a lag of a few pixels before panning is enabled. When panning is enabled, there will be a "Exit Pan" button on the upper right side of the screen. Tap the button to go back to the current position.
  • The map can display a Navigation Window and Altitude Profile Window. Select menu item Tools, Configuration and then Map Properties to enable these windows. You can also get to the Map Properties from the Map Window by using the Tap and Hold on the Map Window, selecting Map Settings and then Map Properties.

Keys while in the Map Window

  • Joystick Up - Zoom In
  • Joystick Down - Zoom Out

How to connect to my GPS receiver?
Make sure that your GPS receiver is connected to your Pocket PC and function. Also take note of which port the GPS is connected to. To connected to the GPS receiver, select the Toots-->GPS-->Connect-->Serial Port (Comm Port) menu item. This will bring up a Communication Settings Dialog where you can select the port and baud rate of your GPS.

How to record and play back NMEA data?
Select menu item Tools, GPS, Log and then start to start to record NMEA data. To play back NMEA data, select menu item Tools, GPS, Connect, Logged NMEA file and then select the NMEA file to play back.

Manufacture Specific GPS Initialization

Some GPS receivers may have some of the NMEA sentences disabled. By selecting the manufacture or GPS chip-set that matches your GPS. BeeLineGPS will send an initialization to the GPS and enable the GPGGA, GPGSA, GPGSV and GPRMC NMEA sentences.

For more information about NMEA, please visit www.nmea.org.
