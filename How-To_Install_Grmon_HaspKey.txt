Install Grmon HASP Key
=======================

GRMON is licensed using a HASP HL USB hardware key. 
HASP HL license drivers can be downloaded from from the Safenet site.


HASP HL license Drivers
https://sentinelcustomer.gemalto.com/sentineldownloads/?s=&c=End+User&p=HASP+HL&o=all&t=Runtime+%26+Device+Driver&l=all

$]# ls

$]# tar zxvf Sentinel_LDK_RedHat_and_SuSE_RPM_Run-time_Installer.tar.gz
